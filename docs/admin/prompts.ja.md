# プロンプト

パス：`/admin/prompts`

アクセス権限：`super_admin`、または「プロンプト」（`prompts`）機能を付与されたメンバー。

---

## このページの目的

各 AI 機能が使うシステムプロンプトを設定します。ここで設定するのは**組織レベル**の上書きで、空欄の場合はシステム内蔵のデフォルトが使われます。

設定できるプロンプト：

| 項目 | 用途 |
|------|------|
| 名刺認識 | Bot／Web で名刺をアップロードする際の OCR 認識 |
| タスク解析 | 自然言語をタスクに解析 |
| メール生成 | AI によるメール本文生成（「ユーザー編集可」と表示） |
| ドキュメント生成 | AI によるドキュメント生成 |
| 会議解析 | 会議メモの解析 |

---

## 手順

1. 対象項目のテキストボックスにカスタムプロンプトを入力します。
2. 「保存」をクリック。**空欄のまま保存**すると上書きが削除され、システムデフォルトに戻ります。
3. 「デフォルトに戻す」を押すとテキストボックスがクリアされます（保存するまで反映されません）。
4. ある項目がデフォルト使用中（ボックスが空）のとき、内蔵のデフォルトプロンプトが下に参考表示されます。

---

## 注意事項

- プロンプトの変更はその機能の AI 出力品質に影響します。小さく調整して結果を確認してください。
- 「メール生成」はユーザー編集可と表示され、その他はシステムレベルの設定です。
