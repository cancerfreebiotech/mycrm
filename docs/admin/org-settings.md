# 組織設定與品牌

位置：`/admin/org-settings`（僅 super_admin）

管理全公司共用的組織資訊與電子報品牌，不需改程式或重新部署。

## 可設定項目

| 項目 | 用途 |
|------|------|
| 組織名稱 | 系統顯示的公司名稱 |
| 允許登入的 Email 網域 | 後端登入網域驗證來源（逗號分隔多網域） |
| 電子報 Logo 網址 | 電子報骨架的 Logo |
| 電子報回信 Email | 電子報的 reply-to |
| 公司網站 / Facebook / LinkedIn | 電子報頁尾的社群連結 |

## 使用方式

1. 修改欄位後按「儲存」。
2. 設定有 **60 秒快取**，儲存後請重新整理頁面以套用。
3. 欄位留空代表使用系統預設值（顯示於欄位提示）。

> 所有變更都會寫入稽核日誌。
