# 管理員

> 此區功能僅限 `super_admin` 角色可見與存取。

以下功能由 Super Admin 統一管理：

| 功能 | 說明 |
|------|------|
| 標籤管理 | 新增、編輯、刪除聯絡人標籤 |
| 未指派筆記 | 查看尚未綁定聯絡人的 Bot 筆記 |
| 郵件範本 | 管理 AI 產生郵件的提示詞範本 |
| AI 模型 | 設定 AI 端點與模型 |
| 使用者管理 | 管理成員帳號與角色 |
| 報表 | 查看業務活動統計報表 |
| 國家管理 | 維護國家清單（ISO 代碼、多語系名稱、旗幟 emoji、啟用狀態） |
| 電子報 | 訂閱者管理 / 電子報編輯 / SendGrid 寄送 / PDF 匯出 / RSS 給 Substack 抓草稿 |
