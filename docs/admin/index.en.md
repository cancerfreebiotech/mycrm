# Admin

> This area is only visible and accessible to users with the `super_admin` role.

The following features are managed exclusively by Super Admin:

| Feature | Description |
|---------|-------------|
| Tag Management | Add, edit, and delete contact tags |
| Unassigned Notes | View Bot notes not yet linked to any contact |
| Email Templates | Manage prompt templates for AI-generated emails |
| AI Models | Configure AI endpoints and models |
| User Management | Manage member accounts and roles |
| Reports | View business activity statistics reports |
| Country Management | Maintain the country list (ISO code, multilingual names, flag emoji, enabled status) |
| Newsletter | Subscriber management / campaign editor / SendGrid sending / PDF export / RSS feed for Substack auto-drafts |
