# 提示詞

路徑：`/admin/prompts`

存取權限：`super_admin`，或已被授權「提示詞」（`prompts`）功能的成員。

---

## 這頁在做什麼

設定各項 AI 功能使用的系統提示詞（system prompt）。這裡設定的是**組織層級**的覆寫；留空則使用系統內建預設。

可設定的提示詞：

| 項目 | 用途 |
|------|------|
| 名片辨識 | Bot／Web 上傳名片時的 OCR 辨識 |
| 任務解析 | 自然語言解析成任務 |
| 郵件產生 | AI 產生郵件內容（標示為「使用者可編輯」） |
| 文件產生 | AI 產生文件 |
| 會議解析 | 會議紀錄解析 |

---

## 操作步驟

1. 在對應項目的文字框輸入自訂提示詞。
2. 點「儲存」。**內容留空後儲存**等於刪除覆寫，改回使用系統預設。
3. 點「還原預設」會把文字框清空（尚未儲存前不會生效）。
4. 當某項目使用預設（文字框為空）時，下方會顯示系統內建的預設提示詞供參考。

---

## 注意事項

- 修改提示詞會影響該功能的 AI 產出品質，建議小幅調整後觀察結果。
- 「郵件產生」標示為使用者可編輯，其餘為系統性設定。
