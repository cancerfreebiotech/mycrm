# 監査ログ

パス：`/admin/audit-log`（`super_admin` のみ。v7.4.0 で追加）

---

## このページの目的

システム内の「機微／特権操作」（例：ロール変更、削除など）の監査ログを確認し、事後の追跡に使います。データは `admin_actions` テーブルから取得し、1 ページ 20 件で前後ページ移動でき、右上の「更新」で再読み込みします。

---

## テーブルの列

| 列 | 説明 |
|------|------|
| 時刻 | 操作が発生した時刻（台北タイムゾーン） |
| 操作者 | 実行者の email |
| アクション | 操作の種類（読みやすいラベルに翻訳） |
| 対象 | 操作された対象（連絡先やユーザー ID など） |
| 詳細 | 追加データ。「展開」で JSON 表示 |

---

## 注意事項

- 本ページは super_admin のみ利用でき、バックエンド API でも権限を再検証します。
- これは閲覧専用のログで、ここでレコードを編集・削除することはできません。
