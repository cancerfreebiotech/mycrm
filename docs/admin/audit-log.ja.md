# 監査ログ

パス：`/admin/audit-log`（`super_admin` のみ。v7.4.0 で追加）

---

## このページの目的

システム内の「機微／特権操作」（例：ロール変更、削除など）の監査ログを確認し、事後の追跡に使います。データは `admin_actions` テーブルから取得し、1 ページ 20 件で前後ページ移動でき、操作者／アクション／日付での絞り込みと CSV エクスポートに対応します。右上の「更新」で再読み込みします。

---

## テーブルの列

| 列 | 説明 |
|------|------|
| 時刻 | 操作が発生した時刻（台北タイムゾーン） |
| 操作者 | 実行者の email |
| アクション | 操作の種類（読みやすいラベルに翻訳） |
| 対象 | 操作された対象（連絡先やユーザー ID など） |
| 詳細 | 追加データ。「展開」で JSON 表示 |

---

## 絞り込みとエクスポート

上部のフィルターバーで範囲を絞り込めます。いずれかの条件を変更すると即座に反映されます：

| フィルター | 説明 |
|------|------|
| 操作者 | email キーワードで実行者を検索 |
| アクション | 19 種すべての特権操作を含むドロップダウン（下記参照） |
| 開始 / 終了日 | **台北タイムゾーン**の日付境界で絞り込み |

- フィルターを適用すると「フィルターをクリア」が表示されます。
- 「CSV エクスポート」は現在の絞り込み結果を出力します（最大 5,000 件、UTF-8 BOM 付きで Excel が日本語・中国語を正しく開けます）。

### 絞り込めるアクション

v7.9.5 以降、19 種の特権操作すべてに読みやすいラベルが付き、バックエンドの書き込み箇所と 1 対 1 で対応します：MFA リセット、Telegram ID 設定、メンテナンスモード切り替え、連絡先の永久削除、一括永久削除、MCP トークン作成、MCP トークン失効、Email 復旧の適用、Hunter 設定変更、Webhook 設定、リリース通知の送信、連絡先の統合、DSAR データ照会、アカウント状態設定、ニュースレター送信、GDPR データエクスポート、ロール設定、機能権限設定、組織設定変更。

---

## 注意事項

- 本ページは super_admin のみ利用でき、バックエンド API でも権限を再検証します。
- これは閲覧専用のログで、ここでレコードを編集・削除することはできません。
