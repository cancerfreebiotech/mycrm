# MCP 活動紀錄

路徑：`/admin/mcp-activity`（僅 `super_admin`）

---

## 這頁在做什麼

檢視外部 AI agent（透過 MCP 權杖）對系統執行的操作紀錄，用來稽核 agent 做了什麼。預設載入最近 100 筆，可點右上「重新整理」重新載入。

頂端有三個統計卡：總數、成功、失敗。

---

## 篩選與檢視

- **工具**：只看特定工具（tool）的操作。
- **狀態**：全部 / 只看成功 / 只看失敗。

表格欄位：時間、工具、狀態（ok／fail）、權杖、操作者（actor）、IP hash、以及可展開查看的參數／錯誤訊息。點「展開」會顯示該筆的呼叫參數（JSON），失敗者另會顯示錯誤訊息。

---

## 注意事項

- 從「MCP 權杖」頁點某權杖的活動連結進來時，會以該權杖的 `token_id` 預先過濾。
- 本頁僅 super_admin 可用。
