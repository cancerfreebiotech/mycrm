# 稽核日誌

路徑：`/admin/audit-log`（僅 `super_admin`；v7.4.0 新增）

---

## 這頁在做什麼

檢視系統中「敏感／特權操作」的稽核紀錄（例如角色變更、刪除等），供事後追查。資料來自 `admin_actions` 表，每頁 20 筆，可上下頁翻閱；右上「重新整理」可重新載入。

---

## 表格內容

| 欄位 | 說明 |
|------|------|
| 時間 | 操作發生時間（台北時區） |
| 操作者 | 執行者的 email |
| 動作 | 操作類型（會翻譯成可讀標籤） |
| 對象 | 被操作的目標（如聯絡人或使用者 ID） |
| 詳情 | 額外資料，點「展開」以 JSON 檢視 |

---

## 注意事項

- 本頁僅 super_admin 可用，後端 API 也會再次驗證權限。
- 這裡是唯讀紀錄，不能在此修改或刪除紀錄。
