# フィードバック管理

パス：`/admin/feedback`（`super_admin` のみ。サイドバーは super_admin にのみ表示）

---

## このページの目的

ユーザーが「フィードバック」から送信した提案や不具合報告を確認・処理し、対応状態を更新します。タイトル横に総件数が表示され、リストは送信日時の新しい順です。

---

## リストの内容

各項目はカード表示：

- **種別**バッジ：バグ（赤）または機能要望（紫）。
- タイトル、送信者、送信日。
- **状態**バッジ：未対応 / 対応中 / 完了 / 対応しない。

カードをクリックすると展開し、完全な説明と、送信者が添付したスクリーンショット（あれば。スクリーンショットはプライベート bucket に保存され、署名付き URL で表示）を確認できます。

---

## 状態の更新

展開したカードの下部で状態ボタンをクリックすると、未対応・対応中・完了・対応しない のいずれかに切り替わります。現在の状態のボタンは選択済みとして表示され、再クリックはできません。

---

## 注意事項

- 本ページは super_admin のみ利用できます。
- 状態変更は即時保存されます。保存に失敗した場合はエラーメッセージが表示されます。
