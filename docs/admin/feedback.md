# 意見回饋管理

路徑：`/admin/feedback`（僅 `super_admin`；側邊欄僅對 super_admin 顯示）

---

## 這頁在做什麼

檢視並處理使用者透過「意見回饋」送出的建議與問題回報，並更新處理狀態。頁面標題旁顯示總筆數，清單依送出時間新到舊排列。

---

## 清單內容

每筆以卡片呈現：

- **類型**徽章：Bug（紅）或功能建議（紫）。
- 標題、送出者、送出日期。
- **狀態**徽章：待處理 / 處理中 / 已完成 / 不修復。

點卡片可展開，看到完整描述，以及送出者附上的截圖（若有；截圖存於私有 bucket，以簽章連結顯示）。

---

## 更新狀態

在展開的卡片下方，點狀態按鈕即可切換為：待處理、處理中、已完成、不修復。目前狀態的按鈕會標示為選中且不可重複點。

---

## 注意事項

- 本頁僅 super_admin 可用。
- 狀態變更即時儲存，若儲存失敗會顯示錯誤訊息。
