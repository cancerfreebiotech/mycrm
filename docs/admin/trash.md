# 回收區

路徑：`/admin/trash`

存取權限：實際上僅 `super_admin`。本頁除了 PermissionGate「回收區」（`trash`）外，另內建 super_admin 檢查；即使被授權「回收區」的一般成員進入，也會看到「無權限存取此頁面」。

---

## 這頁在做什麼

聯絡人被刪除後不會立即消失，而是先移到回收區，可**還原**或由 super_admin **永久刪除**。

列表以表格呈現：勾選框、姓名（點擊看詳情）、公司、刪除者、刪除時間。頂端有警示橫幅提示目前回收區筆數。

---

## 操作

| 動作 | 行為 |
|------|------|
| 還原 | 把聯絡人移回正式清單（需確認）。 |
| 永久刪除 | 徹底刪除單一聯絡人（需確認）。 |
| 刪除選取 | 勾選多筆後批次永久刪除。 |
| 清空全部 | 一次永久刪除回收區內全部聯絡人（需確認）。 |

點姓名可開啟詳情視窗，檢視完整欄位、標籤、名片圖片與最近的互動紀錄（最多 10 筆），視窗內也可直接還原或永久刪除。

---

## 注意事項

- **永久刪除不可復原**，破壞性操作皆需二次確認。
- 名片圖片存於私有 bucket，詳情視窗顯示的是簽章短效連結。
