# MCP 權杖

路徑：`/admin/mcp-tokens`（僅 `super_admin`）

---

## 這頁在做什麼

管理外部 AI agent（透過 MCP）存取本系統時使用的存取金鑰。以表格列出所有已核發的權杖，欄位包含：名稱／用途、指派對象、Scopes、上次使用、到期、狀態（啟用／停用／過期）。右上有前往「MCP 活動」的連結。

---

## 核發權杖

點「核發權杖」開啟視窗，填寫：

- **名稱**（用途）與**說明**。
- **指派對象**：選一位使用者，agent 的操作會以該使用者身分記錄。
- **Scopes**：勾選允許的權限，讀取類（contacts / newsletter / tags）與寫入類（contacts / notes / newsletter）。若某 scope 超出指派對象在系統中的功能授權，會跳出警告，需確認才繼續。
- **允許任何 actor**：選填。
- **有效期**：永久 / 1 年 / 30 天 / 24 小時。

核發後**明文金鑰只顯示一次**，請立即複製保存；關閉後無法再看到。

---

## 管理現有權杖

每列可：前往該權杖的活動紀錄、停用／啟用、刪除（皆需確認）。停用或過期的權杖無法再使用。

---

## 注意事項

- 明文金鑰僅在核發當下顯示一次，遺失只能刪除重發。
- 授予的 scope 應與指派對象在系統中的權限相符；系統會對超出授權的 scope 提出警告。
