# 管理者

> このエリアは `super_admin` ロールを持つユーザーのみが閲覧・アクセスできます。

以下の機能は Super Admin が一元管理します：

| 機能 | 説明 |
|------|------|
| タグ管理 | 連絡先タグの追加・編集・削除 |
| 未割り当てメモ | 連絡先に紐付いていない Bot メモの確認 |
| メールテンプレート | AI によるメール生成のプロンプトテンプレートを管理 |
| AI モデル | AI エンドポイントとモデルの設定 |
| ユーザー管理 | メンバーアカウントとロールの管理 |
| レポート | 営業活動の統計レポートを表示 |
| 国管理 | 国リストの管理（ISO コード・多言語名・国旗 emoji・有効状態） |
