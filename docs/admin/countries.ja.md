# 国管理

パス：`/admin/countries`

アクセス権限：`super_admin`、または「国管理」（`countries`）機能を付与されたメンバー。

---

## このページの目的

システムで利用できる国リストを管理し、連絡先の国設定で選べるようにします。テーブル表示で、列は：ISO コード、国旗 emoji、中国語名、英語名、日本語名、状態。見出しをクリックするとコード／中国語名／英語名で並び替えできます。

---

## 国の追加

右上の「国を追加」でフォームを開きます：

- **コード**：2 文字の ISO 3166-1 コード（自動で大文字化、英字のみ）。よく使うコード（例：TW、JP、US）を入力すると、中国語／英語／日本語名と国旗が**自動入力**され、編集も可能です。
- **中国語名**と**英語名**は必須。日本語名と国旗 emoji は任意。

---

## 編集と状態

- **編集**：行の鉛筆アイコンで名称と国旗を変更できます。コードは作成後変更できません。
- **有効／無効**：状態バッジをクリックで切り替え。無効の国はメニューに表示されませんが、データは保持されます。
- **削除**：ゴミ箱アイコンをクリックし、インラインで確認して削除します。

---

## 注意事項

- コードは大文字英字 2 文字である必要があり、そうでない場合はフォーマットエラーで拒否されます。
- 名称は 3 言語のフィールドがあります。各言語の UI で正しく表示されるよう、3 言語すべて入力することを推奨します。
