# MCP アクティビティログ

パス：`/admin/mcp-activity`（`super_admin` のみ）

---

## このページの目的

外部 AI エージェントが（MCP トークン経由で）システムに対して実行した操作ログを確認し、エージェントが何をしたかを監査します。既定では直近 100 件を読み込み、右上の「更新」で再読み込みできます。

上部に 3 つの統計カード：総数、成功、失敗。

---

## 絞り込みと表示

- **ツール**：特定のツール（tool）の操作のみ表示。
- **状態**：すべて / 成功のみ / 失敗のみ。

テーブルの列：時刻、ツール、状態（ok／fail）、トークン、操作者（actor）、IP ハッシュ、展開可能な引数／エラーメッセージ。「展開」をクリックするとその項目の呼び出し引数（JSON）が表示され、失敗の場合はエラーメッセージも表示されます。

---

## 注意事項

- 「MCP トークン」ページから特定トークンのアクティビティリンク経由で来ると、そのトークンの `token_id` で事前に絞り込まれます。
- 本ページは super_admin のみ利用できます。
