# ユーザー管理

パス：`/admin/users`（super_admin のみ表示）

---

## ユーザー一覧

システムにログインしたすべてのメンバーを表示します。表示されるカラム：

| カラム | 説明 |
|--------|------|
| 表示名 | ユーザーが設定した名前 |
| Email | Microsoft アカウント（主キー） |
| ロール | member / super_admin |
| Telegram ID | 紐付けられた Telegram アカウント |
| AI モデル | 個人で使用している AI モデル |
| 参加日時 | 初回ログイン日時 |

---

## ロール管理

ユーザー一覧のロールボタンをクリックして切り替えます：

- `member` → 一般ユーザー
- `super_admin` → 管理機能にアクセス可能

> 注意：すべての管理者が降格されないよう、少なくとも 1 名の super_admin を維持してください。

---

## 注意事項

- ユーザーアカウントは**初回ログイン時に自動作成**されます。手動で事前作成することはできません
- `@cancerfree.io` ドメインの Microsoft アカウントのみログインできます
- ユーザーの削除：現時点では Supabase Dashboard から DB を直接操作する必要があります
