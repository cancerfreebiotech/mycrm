# ゴミ箱

パス：`/admin/trash`

アクセス権限：実質的に `super_admin` のみ。本ページは「ゴミ箱」（`trash`）の PermissionGate に加え、super_admin チェックを内蔵しています。「ゴミ箱」を付与された一般メンバーが開いても「このページにアクセスする権限がありません」と表示されます。

---

## このページの目的

削除された連絡先はすぐには消えず、まずゴミ箱に移動します。ここで**復元**するか、super_admin が**完全削除**できます。

リストはテーブル表示：チェックボックス、氏名（クリックで詳細）、会社、削除者、削除日時。上部の警告バナーに現在のゴミ箱件数が表示されます。

---

## 操作

| 操作 | 動作 |
|------|------|
| 復元 | 連絡先を正式リストに戻す（確認が必要）。 |
| 完全削除 | 単一の連絡先を完全に削除（確認が必要）。 |
| 選択を削除 | 複数行を選択して一括で完全削除。 |
| すべて削除 | ゴミ箱内の全連絡先を一度に完全削除（確認が必要）。 |

氏名をクリックすると詳細ダイアログが開き、全項目・タグ・名刺画像・直近の交流記録（最大 10 件）を確認できます。ダイアログ内から直接復元・完全削除も可能です。

---

## 注意事項

- **完全削除は取り消せません**。破壊的操作はすべて再確認が必要です。
- 名刺画像はプライベート bucket に保存され、詳細ダイアログには短時間有効の署名付き URL が表示されます。
