# 組織設定とブランド

場所：`/admin/org-settings`（super_admin のみ）

コード変更や再デプロイなしで、全社共通の組織情報とニュースレターのブランドを管理します。

## 設定項目

| 項目 | 用途 |
|------|------|
| 組織名 | システムに表示される会社名 |
| ログイン許可メールドメイン | バックエンドのログインドメイン検証元（カンマ区切り） |
| ニュースレターのロゴ URL | ニュースレター骨組みのロゴ |
| ニュースレターの返信先メール | ニュースレターの reply-to |
| 会社ウェブサイト / Facebook / LinkedIn | ニュースレターフッターのソーシャルリンク |

## 使い方

1. 項目を編集して「保存」をクリック。
2. 設定は **60 秒キャッシュ**されます。保存後はページを更新して反映してください。
3. 空欄はシステムのデフォルト値を使用します（項目のヒントに表示）。

> すべての変更は監査ログに記録されます。
