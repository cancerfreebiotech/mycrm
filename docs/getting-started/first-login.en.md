# First Login

## Login Method

myCRM uses **Microsoft Account (Azure AD) Single Sign-On**, restricted to the `@cancerfree.io` domain.

1. Go to [https://mycrm-vert.vercel.app](https://mycrm-vert.vercel.app)
2. Click "Sign in with Microsoft Account"
3. Enter your company Email and password
4. Approve the application authorization
5. On your first login, the system requires you to complete **MFA (two-factor authentication) setup** (see below)

> After your first login, the system automatically creates your account in the `users` table (role defaults to `member`).

---

## Set Up MFA (Two-Factor Authentication)

For account security, **every user must complete MFA (two-factor authentication)**. After your first login, the system automatically redirects you to the MFA setup page, and no feature is usable until it is done.

1. Install an authenticator app on your phone (e.g. Google Authenticator, Microsoft Authenticator, 1Password)
2. Scan the on-screen QR code with the app (or manually enter the secret key shown below it)
3. Enter the **6-digit code** generated by the app to finish enrollment
4. Once enrolled, you can use the system normally

> On every subsequent login, the system asks for the current 6-digit code from your authenticator before letting you in.

- To switch or re-enroll your authenticator: top-right corner → **Settings** → MFA section.
- If you lose your phone or cannot get a code: contact a super_admin to reset your MFA on the **User Management** page, then sign in again to re-enroll.

## Linking Telegram

To use Telegram Bot features (business card scanning, task management), you need to link your Telegram account first:

1. Search for **`@userinfobot`** on Telegram, send any message to get your Telegram ID (a number, e.g. `123456789`)
2. Log in to myCRM → top-right corner → **Personal Settings** → enter your Telegram ID and save
3. Search for **`@CF_CRMBot`** (CancerFreeCRM) on Telegram and start a conversation

> After linking, send `/help` to see all available commands.

---

## Linking Microsoft Teams Bot

To receive task notifications and mark tasks complete directly in Teams, complete the linking once:

1. Search for the Bot name in Microsoft Teams (provided by your administrator)
2. Open a **1-on-1 chat**
3. Send any message (e.g. `help`)
4. The Bot automatically resolves your AAD account and links it to your CRM account

When linking is successful, the Bot replies:
```
📋 myCRM Bot (Linked: your.email@cancerfree.io)
Task notifications will be sent here automatically.
```

> Linking only needs to be done once. After that, you will automatically receive Teams notification cards when tasks are assigned to you.

## Language Settings

myCRM supports three languages: **Traditional Chinese**, **English**, **Japanese**

How to switch (either method works):
- Click the 🌐 icon in the top-right corner of the Header → select language
- **Personal Settings** → Language selection section

Your language preference is saved to your account and applied automatically on next login.

## Interface Overview

After login, the sidebar is on the left. The top section holds core features every member can use; items below the divider are advanced features that require a grant. A floating AI Assistant button sits at the bottom-right of every page.

**Visible to all members (core features):**

| Item | Description |
|------|-------------|
| Dashboard | Dashboard and stats |
| Contacts | Business-card contact list |
| Pending Cards | Cards awaiting your review after AI recognition |
| My Failed Scans | Cards you scanned that failed recognition |
| Photos | Card and event photos |
| Search Notes | Search contact notes |
| Tasks | Task list |
| Reports | Statistics reports |
| Email Campaigns | Email marketing campaigns |
| Settings | Profile, language, MFA, Telegram linking |
| Documentation | This documentation |
| 💬 Feedback | Send feedback |

> On mobile the sidebar is a hamburger menu; on desktop it expands on hover, and you can pin it collapsed with the arrow at the bottom-left.

**Advanced features (grant required):**

Items below the divider (Tag Management, Unassigned Notes, Email Templates, Prompt Management, Country Management, Newsletter, Failed Scans, Duplicates, Shared Emails, Email Recovery, CamCard Import, Trash, User Management) require a super_admin to grant them per feature on the **User Management** page before you can use them.

**super_admin only:**

AI Model Management, System Health, MCP Tokens, MCP Activity, and Feedback Management are visible only to super_admin.

## Role Descriptions

myCRM has only two roles:

| Role | Description |
|------|-------------|
| `member` | General user. Can use all core features (Dashboard, Contacts, Pending Cards, My Failed Scans, Photos, Search Notes, Tasks, Reports, Email Campaigns, Settings, Documentation, Feedback, AI Assistant); advanced features require a separate grant. |
| `super_admin` | System administrator. Has every feature, and can manage users, assign feature grants, reset other users' MFA, and manage AI models and system settings. |

Advanced features use **per-feature grants (granted features)**: a super_admin enables the needed features (e.g. Newsletter, Tag Management, User Management) for an individual member on the **User Management** page, and that member can then use the feature without any role change.
