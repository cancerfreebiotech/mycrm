# 初回ログイン

## ログイン方法

myCRM は **Microsoft アカウント（Azure AD）シングルサインオン**を使用しており、`@cancerfree.io` ドメインのアカウントのみ利用できます。

1. [https://mycrm-vert.vercel.app](https://mycrm-vert.vercel.app) にアクセスします
2. 「Microsoft アカウントでサインイン」をクリックします
3. 会社のメールアドレスとパスワードを入力します
4. アプリケーションの認可に同意します

> 初回ログイン後、システムは自動的に `users` テーブルにアカウントを作成します（ロールのデフォルトは `member`）。

---

## Telegram の連携

Telegram Bot 機能（名刺スキャン・タスク管理）を使用するには、先に Telegram アカウントを連携する必要があります：

1. Telegram で **`@userinfobot`** を検索し、任意のメッセージを送信して Telegram ID（数字のみ、例：`123456789`）を取得します
2. myCRM にログイン → 右上角 → **個人設定** → Telegram ID を入力して保存します
3. Telegram で **`@CF_CRMBot`**（CancerFreeCRM）を検索して会話を開始します

> 連携後、`/help` を送信すると利用可能なコマンド一覧が表示されます。

---

## Microsoft Teams Bot の連携

タスク通知を受け取り、Teams で直接タスクを完了にするには、一度連携操作を行う必要があります：

1. Microsoft Teams で Bot 名を検索します（管理者から提供）
2. **1 対 1 チャット**を開きます
3. 任意のメッセージを送信します（例：`help`）
4. Bot が自動的に AAD アカウントを解析し、CRM アカウントと紐付けます

連携が成功すると Bot が次のように返信します：
```
📋 myCRM Bot（連携済み：your.email@cancerfree.io）
タスク通知はここに自動送信されます。
```

> 連携は一度行うだけで済みます。以降、タスクが割り当てられると自動的に Teams 通知カードが届きます。

## 言語設定

myCRM は 3 つの言語に対応しています：**繁体字中国語**・**English**・**日本語**

切り替え方法（どちらでも可）：
- ヘッダー右上の 🌐 アイコンをクリック → 言語を選択
- **個人設定** → 言語選択セクション

言語設定はアカウントに保存され、次回ログイン時に自動的に適用されます。

## ロールの説明

| ロール | 説明 |
|--------|------|
| `member` | 一般ユーザー。ホーム・連絡先・メモ・タスク・設定・ドキュメントを使用できます |
| `super_admin` | 追加アクセス：タグ管理・未割り当てメモ・メールテンプレート・AI モデル・ユーザー管理・レポート |

ロールは super_admin が「ユーザー管理」ページで設定します。
