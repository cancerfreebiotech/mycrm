# 第一次登入

## 登入方式

myCRM 使用 **Microsoft 帳號（Azure AD）單一登入**，限 `@cancerfree.io` 網域。

1. 前往 [https://mycrm.vercel.app](https://mycrm.vercel.app)
2. 點擊「使用 Microsoft 帳號登入」
3. 輸入公司 Email 和密碼
4. 同意應用程式授權

> 首次登入後，系統會自動在 `users` 資料表建立你的帳號（role 預設為 `member`）。

---

## 綁定 Telegram

要使用 Telegram Bot 功能（名片掃描、任務管理），需要先綁定你的 Telegram 帳號：

1. 登入 Web Dashboard 後，前往右上角 → **個人設定**
2. 找到「Telegram ID」欄位
3. 前往 Telegram，找到 Bot（搜尋 `@YourBotName`）
4. 發送 `/start` 指令，Bot 會顯示你的 Telegram ID
5. 將 ID 填入個人設定並儲存

---

## 語言設定

myCRM 支援三種語言：**繁體中文**、**English**、**日本語**

切換方式（任一皆可）：
- Header 右上角點擊 🌐 圖示 → 選擇語言
- **個人設定** → 語言選擇區塊

語言偏好儲存在你的帳號，下次登入自動套用。

## 角色說明

| 角色 | 說明 |
|------|------|
| `member` | 一般使用者，可管理聯絡人、任務、筆記 |
| `super_admin` | 可額外存取標籤管理、使用者管理、AI 模型、報表管理 |

角色由 super_admin 在「使用者管理」頁面設定。
