# 第一次登入

## 登入方式

myCRM 使用 **Microsoft 帳號（Azure AD）單一登入**，限 `@cancerfree.io` 網域。

1. 前往 [https://mycrm-vert.vercel.app](https://mycrm-vert.vercel.app)
2. 點擊「使用 Microsoft 帳號登入」
3. 輸入公司 Email 和密碼
4. 同意應用程式授權
5. 首次登入後，系統會要求你完成 **MFA（雙重驗證）設定**（見下方）

> 首次登入後，系統會自動在 `users` 資料表建立你的帳號（role 預設為 `member`）。

---

## 設定 MFA（雙重驗證）

為了帳號安全，**所有使用者都必須完成 MFA（雙重驗證）**。首次登入後，系統會自動導向 MFA 設定頁，未完成前無法使用任何功能。

1. 在手機安裝一款驗證器 App（如 Google Authenticator、Microsoft Authenticator、1Password 等）
2. 用 App 掃描畫面上的 QR Code（或手動輸入下方顯示的密鑰）
3. 輸入 App 產生的 **6 位數驗證碼**，完成綁定
4. 綁定成功後即可正常使用系統

> 之後每次登入，系統都會要求輸入驗證器當下的 6 位數驗證碼才能進入。

- 想更換或重新綁定驗證器：右上角 → **個人設定** → MFA 區塊管理。
- 手機遺失或無法取得驗證碼：請聯絡 super_admin，於 **使用者管理** 頁面為你重設 MFA，之後重新登入即可重新綁定。

---

## 綁定 Telegram

要使用 Telegram Bot 功能（名片掃描、任務管理），需要先綁定你的 Telegram 帳號：

1. 在 Telegram 搜尋 **`@userinfobot`**，發送任何訊息，取得你的 Telegram ID（純數字，如 `123456789`）
2. 登入 myCRM → 右上角 → **個人設定** → 將 Telegram ID 填入並儲存
3. 前往 Telegram 搜尋 **`@CF_CRMBot`**（CancerFreeCRM），開始對話即可使用

> 綁定完成後可發送 `/help` 查看所有可用指令。

---

## 綁定 Microsoft Teams Bot

要接收任務通知並在 Teams 直接標記完成，需完成一次綁定：

1. 在 Microsoft Teams 搜尋 Bot 名稱（由管理員提供）
2. 開啟 **1-on-1 聊天**
3. 傳送任何訊息（例如 `help`）
4. Bot 自動解析你的 AAD 帳號並與 CRM 帳號綁定

綁定成功時 Bot 會回覆：
```
📋 myCRM Bot（已綁定：your.email@cancerfree.io）
任務通知會自動傳送到這裡。
```

> 綁定僅需操作一次，之後任務指派時會自動收到 Teams 通知卡片。

## 語言設定

myCRM 支援三種語言：**繁體中文**、**English**、**日本語**

切換方式（任一皆可）：
- Header 右上角點擊 🌐 圖示 → 選擇語言
- **個人設定** → 語言選擇區塊

語言偏好儲存在你的帳號，下次登入自動套用。

## 介面說明

登入後左側為側邊欄，上半部是所有成員都能使用的核心功能，分隔線以下為需授權的進階功能。頁面右下角有 AI 助理浮動按鈕，任何頁面都可開啟。

**所有成員可見（核心功能）：**

| 項目 | 說明 |
|------|------|
| 總覽 | 儀表板與統計 |
| 聯絡人 | 名片聯絡人清單 |
| 待審核名片 | AI 辨識後待你確認的名片 |
| 我的失敗辨識 | 你自己掃描但辨識失敗的名片 |
| 相簿 | 名片與活動照片 |
| 筆記搜尋 | 搜尋聯絡人筆記 |
| 任務 | 任務清單 |
| 報表 | 統計報表 |
| 郵件活動 | Email 行銷活動 |
| 個人設定 | 個人資料、語言、MFA、Telegram 綁定 |
| 使用說明 | 本說明文件 |
| 💬 回饋 | 意見回饋 |

> 側邊欄在手機為漢堡選單，桌機滑鼠移入自動展開，可用左下角箭頭固定收合。

**進階功能（需授權）：**

分隔線以下的項目（Tag 管理、未歸類筆記、郵件範本、Prompt 管理、國家管理、電子報、失敗掃描、重複聯絡人、共用 Email、Email 復活、CamCard 匯入、回收區、使用者管理）需由 super_admin 在**使用者管理**中逐項授權後才能使用。

**super_admin 專屬：**

組織設定、AI 模型管理、系統健康、MCP Tokens、MCP 活動、稽核日誌、回饋管理，僅 super_admin 可見。

## 角色說明

myCRM 只有兩種角色：

| 角色 | 說明 |
|------|------|
| `member` | 一般使用者。可使用所有核心功能（總覽、聯絡人、待審核名片、我的失敗辨識、相簿、筆記搜尋、任務、報表、郵件活動、個人設定、使用說明、回饋、AI 助理）；進階功能需另外授權。 |
| `super_admin` | 系統管理員。具備全部功能，並可管理使用者、指派進階功能授權、重設他人 MFA、管理 AI 模型與系統設定。 |

進階功能採**逐項授權（granted features）**：super_admin 在**使用者管理**頁面為個別成員開啟所需功能（例如電子報、Tag 管理、使用者管理等），成員即可使用該功能，無需變更角色。
