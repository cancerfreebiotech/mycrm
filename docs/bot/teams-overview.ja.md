# Teams Bot 概要

myCRM Teams Bot はタスク通知と会議スケジュール機能を提供します。タスクが割り当てられると、Microsoft Teams の個人チャットに **Adaptive Card** 通知が届きます。また、自然言語コマンドで会議を設定することもできます。

---

## 機能

| 機能 | 説明 |
|------|------|
| タスク通知 | 新しいタスクが割り当てられると Teams に Adaptive Card を送信 |
| ワンクリック完了 | Web を開かずに Teams で直接「完了にする」ボタンをクリック可能 |
| タスクリンク | カードに「タスク管理へ移動」リンクを含む |
| 会議スケジュール | `/meet`、`/m` で自然言語による会議設定、確認カードからワンクリックでカレンダーに登録 |
| AI モデル | `/ai` で現在使用中の AI モデルを表示 |
| `/help` コマンド | Teams チャットで `help` と入力するとヘルプを表示 |

---

## 通知カードの例

```
┌────────────────────────────────────┐
│ ✅ タスク割り当て通知               │
│                                    │
│ 📌 Q1 業績レポートを整理してください │
│ ⏰ 期限：2026/03/31 18:00          │
│ 👤 割り当て者：陳ディレクター       │
│                                    │
│  [完了にする]  [タスク管理へ移動]  │
└────────────────────────────────────┘
```

「完了にする」をクリックすると、Bot がタスクのステータスを更新して確認メッセージを返信します。

---

## 会議スケジュール（`/meet`）

Bot とのチャットで、自然言語で会議を設定できます。Bot が AI で日時・参加者・場所を解析し、確認カードを返信します。「確認」を押して初めて、Outlook／Teams のカレンダーに実際に登録されます。

```
/meet 明日の午後3時 Luna と製品ミーティング
/m 3/25 午後1時 九州大学の研究室を訪問
→ Bot: ⏳ AI 解析中...
→ Bot: [確認カード]
        📅 製品ミーティング
        🕐 3/25(火) 13:00 – 14:00（台北）
        👤 あなた、Luna
        [確認]  [キャンセル]
→ 「確認」を押す → Bot: ✅ 予定を作成しました！（カレンダーリンク付き）
```

- 参加者は組織メンバー一覧から名前／Email で照合されます。
- 予定の作成には、myCRM の Web で Microsoft ログイン認可を完了している必要があります。
- 「キャンセル」を押すと予定は作成されません。

---

## AI モデル（`/ai`）

チャットで `/ai` と入力すると、Bot が現在使用中の AI モデルを返信します。未設定の場合はシステムのデフォルト（gemini-2.5-flash）が使用されます。

---

## アカウント連携

Bot は**自動連携**の仕組みを採用しており、手動操作は不要です：

1. Teams で Bot 名を検索し、**1 対 1 チャット**を開きます
2. 任意のメッセージを送信します（例：`help`）
3. Bot が Microsoft Graph を通じて AAD アカウントを自動解析 → CRM ユーザーと照合 → 連携完了

連携が成功すると `help` コマンドの応答に以下が表示されます：
```
📋 myCRM Bot（連携済み：your.email@company.com）
タスク通知はここに自動送信されます。
```

> 連携には Teams アカウントのメールアドレスと CRM ログインメールアドレスが同一である必要があります（同じ Microsoft 365 アカウント）。

---

## 制限事項

- Teams Bot は現在**個人チャット**通知のみに対応しています（チャンネルへのアクティブ配信は非対応）
- ユーザーは Teams で Bot との会話を少なくとも 1 回行わないと通知を受け取れません
- セットアップ手順は [Teams Bot 設定](../deployment/teams-setup.md) をご覧ください
