# Teams Bot 概要

myCRM Teams Bot はタスク通知機能を提供します。タスクが割り当てられると、Microsoft Teams の個人チャットに **Adaptive Card** 通知が届きます。

---

## 機能

| 機能 | 説明 |
|------|------|
| タスク通知 | 新しいタスクが割り当てられると Teams に Adaptive Card を送信 |
| ワンクリック完了 | Web を開かずに Teams で直接「完了にする」ボタンをクリック可能 |
| タスクリンク | カードに「タスク管理へ移動」リンクを含む |
| `/help` コマンド | Teams チャットで `help` と入力するとヘルプを表示 |

---

## 通知カードの例

```
┌────────────────────────────────────┐
│ ✅ タスク割り当て通知               │
│                                    │
│ 📌 Q1 業績レポートを整理してください │
│ ⏰ 期限：2026/03/31 18:00          │
│ 👤 割り当て者：陳ディレクター       │
│                                    │
│  [完了にする]  [タスク管理へ移動]  │
└────────────────────────────────────┘
```

「完了にする」をクリックすると、Bot がタスクのステータスを更新して確認メッセージを返信します。

---

## アカウント連携

Bot は**自動連携**の仕組みを採用しており、手動操作は不要です：

1. Teams で Bot 名を検索し、**1 対 1 チャット**を開きます
2. 任意のメッセージを送信します（例：`help`）
3. Bot が Microsoft Graph を通じて AAD アカウントを自動解析 → CRM ユーザーと照合 → 連携完了

連携が成功すると `help` コマンドの応答に以下が表示されます：
```
📋 myCRM Bot（連携済み：your.email@company.com）
タスク通知はここに自動送信されます。
```

> 連携には Teams アカウントのメールアドレスと CRM ログインメールアドレスが同一である必要があります（同じ Microsoft 365 アカウント）。

---

## 制限事項

- Teams Bot は現在**個人チャット**通知のみに対応しています（チャンネルへのアクティブ配信は非対応）
- ユーザーは Teams で Bot との会話を少なくとも 1 回行わないと通知を受け取れません
- セットアップ手順は [Teams Bot 設定](../deployment/teams-setup.md) をご覧ください
