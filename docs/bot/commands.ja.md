# Bot コマンド一覧

| コマンド | 省略形 | 説明 |
|---------|--------|------|
| `/help` | `/h` | 利用可能なすべてのコマンドを表示 |
| `/search [キーワード]` | `/s` | 連絡先を検索（名前/Email のあいまい検索） |
| `/note [名前]` | `/n` | 連絡先にインタラクションメモを追加 |
| `/visit [名前]` | `/v` | 連絡先に訪問記録を追加 |
| `/a [名前]` | — | 連絡先に名刺写真を追加（OCR） |
| `/p [名前]` | — | 連絡先に集合写真を追加 |
| `/email [キーワード]` | `/e` | 連絡先詳細からメールを送信 |
| `/work [説明]` | `/w` | AI が自然言語を解析してタスクを作成 |
| `/tasks` | `/t` | 自分の未処理タスクを一覧表示 |
| `/user` | `/u` | 組織のメンバー全員を一覧表示 |

---

## コマンド詳細

### `/search` — 連絡先検索

```
/search 山田太郎
/s 山田
/s cancerfree
```

検索結果には名刺写真と以下のクイックボタンが表示されます：
- **📋 インタラクションログ** — 最新 5 件を表示、「さらに読み込む」に対応
- **✉️ メール送信** — メール送信フローへ移動
- **📝 メモ** — メモ記録フローへ移動

---

### `/note` / `/n` — インタラクションメモの追加

**方法 1：インタラクティブ**
```
/note または /n
→ Bot: 前回の連絡先に対してですか？（該当する場合）
→ 名前を選択または入力して検索
→ Bot: メモの内容を入力してください
→ 入力: 本日訪問、協力に前向きな姿勢
```

**方法 2：名前を指定してクイック検索**
```
/n 山田太郎
→ Bot: 見つかりました。メモ入力フローへ移動します
```

**連絡先が見つからない場合**：メモは未分類として自動保存されます。後で Web から分類できます。

**名前の省略**：最近操作した連絡先がある場合、名前を省略すると Bot が「○○さんに対してですか？」と確認します。

---

### `/visit` / `/v` — 訪問記録の追加

**方法 1：インタラクティブ**
```
/visit または /v
→ Bot: 前回の連絡先に訪問記録を追加しますか？（該当する場合）
→ 名前を選択または入力して検索
→ Bot: 訪問日時を入力してください（例：2026-03-29 14:00）、またはスキップと入力
→ Bot: 訪問場所を入力してください、またはスキップと入力
→ Bot: 訪問内容を入力してください
```

**方法 2：名前を指定してクイック検索**
```
/v 山田太郎
→ Bot: 見つかりました。日時入力フローへ移動します
```

---

### `/a` — 連絡先に名刺を追加

すでに連絡先が存在し、名刺写真を追加または置き換えたい場合に使用します。送信後、AI OCR が認識を行い、既存データとの差分を表示してユーザーが更新するかどうかを確認します。

**方法 1：前回の連絡先を使用**
```
/a
→ Bot: 前回の連絡先：山田太郎、名刺写真を送信してください
→ 写真を送信
→ AI OCR → 差分を表示 → 確認/キャンセル
```

**方法 2：名前を指定**
```
/a 山田太郎
→ Bot: 見つかりました。名刺写真を送信してください
```

---

### `/p` — 連絡先に集合写真を追加

**方法 1：前回の連絡先を使用**
```
/p
→ Bot: 前回の連絡先：山田太郎、集合写真を送信してください
→ 写真を送信（推奨：長押し → ファイルとして送信すると撮影日時と GPS 情報が保持されます）
```

**方法 2：名前を指定**
```
/p 山田太郎
→ Bot: 見つかりました。集合写真を送信してください
```

---

### `/email` — メール送信

```
/email
→ 連絡先を検索 → 選択
→ 方法を選択：テンプレート / AI 生成
→ 内容を確認 → 送信
```

メールは Microsoft メールボックスから送信されます（Microsoft Graph API の認可が必要）。

---

### `/work` — タスク作成（AI 自然言語解析）

```
/work 来週金曜日までに山田さんに Q1 業績レポートをまとめてもらう
/w 明日の朝に田中さんに電話することを自分にリマインド
/w 鈴木さんと佐藤さんに今月末までに顧客リストを完成させてもらう
```

AI（Gemini）が解析するもの：
- **タイトル**：タスクの主な内容
- **期限**：「明日」「来週金曜日」「今月末」などの自然言語
- **担当者**：システムのメンバー一覧から名前で照合

---

### `/tasks` — 未処理タスクの確認

自分のすべての `pending` タスクを一覧表示します（自分が作成したもの + 自分に割り当てられたもの）。

各タスクにボタンが付きます：
- **✅ 完了** — 完了としてマーク
- **⏭ 延期** — 新しい期限を入力
- **❌ キャンセル** — タスクをキャンセル

---

### `/user` — 組織メンバー一覧

すべてのメンバーの名前・メールアドレス・Telegram ID を表示します（全メンバーが使用可能）。

---

## 名刺の送信（コマンド不要）

Bot との会話に写真を直接送信します：

```
[写真を送信]
→ Bot: 認識中...
→ Bot: 認識結果：
       名前：山田太郎
       会社：ABC 株式会社
       ...
       保存しますか？
→ ✅ 保存 / ❌ 破棄
```

認識結果が誤っている場合は、保存後に Web で編集できます。
