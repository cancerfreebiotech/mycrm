# Bot 指令列表

| 指令 | 縮寫 | 說明 |
|------|------|------|
| `/help` | `/h` | 顯示所有可用指令 |
| `/search [關鍵字]` | `/s` | 搜尋聯絡人（模糊比對姓名/Email） |
| `/note [關鍵字]` | `/n` | 為聯絡人新增互動筆記 |
| `/email [關鍵字]` | `/e` | 從聯絡人詳情發送 Email |
| `/add_back [@姓名]` | `/ab` | 補充現有聯絡人的名片反面 |
| `/work [描述]` | `/w` | AI 解析自然語言，建立任務 |
| `/tasks` | `/t` | 列出你的待處理任務 |
| `/user` | `/u` | 列出所有組織成員 |

---

## 指令詳細說明

### `/search` — 搜尋聯絡人

```
/search 王大華
/s 王
/s cancerfree
```

搜尋結果每筆顯示名片照片，附快速按鈕：
- **📋 互動紀錄** — 查看最新 5 筆，支援「載入更多」
- **✉️ 發信** — 跳轉到 Email 發送流程
- **📝 筆記** — 跳轉到記錄筆記流程

---

### `/note` — 新增互動筆記

**方式一：互動式**
```
/note
→ Bot: 請輸入關鍵字搜尋聯絡人
→ 輸入: 王大華
→ Bot: 顯示搜尋結果，請選擇編號
→ 輸入: 1
→ Bot: 請輸入筆記內容
→ 輸入: 今天拜訪，有合作意願
```

**方式二：快速格式**
```
/note @王大華 今天拜訪，有合作意願
```

**找不到聯絡人時**：筆記自動存為未歸類，之後可從 Web 歸類。

**省略姓名**：若最近有操作過某位聯絡人，可省略姓名，Bot 會提示「是否針對 XXX？」

---

### `/email` — 發送 Email

```
/email
→ 搜尋聯絡人 → 選擇
→ 選擇方式：範本 / AI 生成
→ 確認內容 → 發送
```

發信使用你的 Microsoft 信箱（需完成 Microsoft Graph API 授權）。

---

### `/add_back` — 補充名片反面

```
/add_back @王大華
→ Bot: 請傳送名片反面照片
→ 傳送照片
→ AI 辨識 → 更新聯絡人資料
```

---

### `/work` — 建立任務（AI 自然語言解析）

```
/work 下週五前請陳小明整理 Q1 業績報告
/w 提醒自己明天早上打電話給王大華
/w 請李四和王五本月底前完成客戶名單
```

AI（Gemini）會解析：
- **標題**：任務主要內容
- **截止時間**：「明天」「下週五」「本月底」等自然語言
- **負責人**：從系統成員名單中比對姓名

---

### `/tasks` — 查看待處理任務

列出你的所有 `pending` 任務（我建立的 + 指派給我的）。

每筆任務附按鈕：
- **✅ 完成** — 標記完成
- **⏭ 延後** — 輸入新截止時間
- **❌ 取消** — 取消任務

---

### `/user` — 列出組織成員

顯示所有成員的姓名、Email 和 Telegram ID（所有成員皆可使用）。

---

## 傳送名片（無需指令）

直接在 Bot 對話中傳送圖片：

```
[傳送照片]
→ Bot: 辨識中...
→ Bot: 辨識結果：
       姓名：王大華
       公司：ABC 公司
       ...
       是否存檔？
→ ✅ 存檔 / ❌ 不存檔
```

若辨識結果有誤，存檔後可進入 Web 編輯。
