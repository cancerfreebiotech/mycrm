# Bot 指令列表

| 指令 | 縮寫 | 說明 |
|------|------|------|
| `/help` | `/h` | 顯示所有可用指令 |

## 指令詳細說明

### `/search` — 搜尋聯絡人

```
/search 王大華
/s 王
/s cancerfree
```

搜尋結果每筆顯示名片照片，附快速按鈕：
- **📋 互動紀錄** — 查看最新 5 筆，支援「載入更多」
- **✉️ 發信** — 跳轉到 Email 發送流程
- **📝 筆記** — 跳轉到記錄筆記流程

---

### `/note` / `/n` — 新增互動筆記

**方式一：互動式**
```
/note 或 /n
→ Bot: 要針對上一位聯絡人嗎？（若有）
→ 選擇或輸入姓名搜尋
→ Bot: 請輸入筆記內容
→ 輸入: 今天拜訪，有合作意願
```

**方式二：帶姓名快速搜尋**
```
/n 王大華
→ Bot: 找到後直接進入輸入筆記流程
```

**找不到聯絡人時**：筆記自動存為未歸類，之後可從 Web 歸類。

**省略姓名**：若最近有操作過某位聯絡人，可省略姓名，Bot 會提示「是否針對 XXX？」

---

### `/visit` / `/v` — 新增拜訪紀錄

**方式一：互動式**
```
/visit 或 /v
→ Bot: 要為上一位聯絡人新增拜訪紀錄嗎？（若有）
→ 選擇或輸入姓名搜尋
→ Bot: 請輸入拜訪日期時間（例：2026-03-29 14:00），或輸入「略過」
→ Bot: 請輸入拜訪地點，或輸入「略過」
→ Bot: 請輸入拜訪內容
```

**方式二：帶姓名快速搜尋**
```
/v 王大華
→ Bot: 找到後直接進入輸入日期時間流程
```

---

### `/a` — 為聯絡人新增名片

用於已有聯絡人但想補充/替換名片照片的情況。傳送後 AI OCR 辨識，顯示與現有資料的差異，由使用者確認是否更新。

**方式一：使用上一位聯絡人**
```
/a
→ Bot: 上一位聯絡人：王大華，請傳送名片照片
→ 傳送照片
→ AI OCR → 顯示 diff → 確認/取消
```

**方式二：帶姓名指定**
```
/a 王大華
→ Bot: 找到後請傳送名片照片
```

---

### `/p` — 為聯絡人新增合照

**方式一：使用上一位聯絡人**
```
/p
→ Bot: 上一位聯絡人：王大華，請傳送合照
→ 傳送照片（建議長按 → 以檔案傳送，可保留拍攝時間和 GPS 地點）
```

**方式二：帶姓名指定**
```
/p 王大華
→ Bot: 找到後請傳送合照
```

---

### `/email` — 發送 Email

```
/email
→ 搜尋聯絡人 → 選擇
→ 選擇方式：範本 / AI 生成
→ 確認內容 → 發送
```

發信使用你的 Microsoft 信箱（需完成 Microsoft Graph API 授權）。

---

### `/work` — 建立任務（AI 自然語言解析）

```
/work 下週五前請陳小明整理 Q1 業績報告
/w 提醒自己明天早上打電話給王大華
/w 請李四和王五本月底前完成客戶名單
```

AI（Gemini）會解析：
- **標題**：任務主要內容
- **截止時間**：「明天」「下週五」「本月底」等自然語言
- **負責人**：從系統成員名單中比對姓名

---

### `/tasks` — 查看待處理任務

列出你的所有 `pending` 任務（我建立的 + 指派給我的）。

每筆任務附按鈕：
- **✅ 完成** — 標記完成
- **⏭ 延後** — 輸入新截止時間
- **❌ 取消** — 取消任務

---

### `/user` — 列出組織成員

顯示所有成員的姓名、Email 和 Telegram ID（所有成員皆可使用）。

## 傳送名片（無需指令）

直接在 Bot 對話中傳送圖片：

```
[傳送照片]
→ Bot: 辨識中...
→ Bot: 辨識結果：
       姓名：王大華
       公司：ABC 公司
       ...
       是否存檔？
→ ✅ 存檔 / ❌ 不存檔
```

若辨識結果有誤，存檔後可進入 Web 編輯。

---

## 批次拍照（`/b` + `/done`）

適合一次有很多張要處理的時候（剛從會議回來、堆名片）。流程跟單張同步辨識不一樣：**Bot 立刻收下、不當下辨識**，使用者可以連續拍很多張不用等；後台辨識完成再通知，到 web 上的「待審核名片」（`/contacts/pending`）統一審核。

```
/b
→ Bot: 📦 已進入 batch 模式
[拍照 #1] → Bot: 📥 已收第 1 張
[拍照 #2] → Bot: 📥 已收第 2 張
...
/done
→ Bot: ✅ 共 N 張已送辨識，完成後通知你
（背景辨識中…）
→ Bot: ✅ 已完成辨識 N/M 張，前往審核 → /contacts/pending
```

中途想退出：`/cancel`，已收的照片**不會刪除**，留在 pending 區可在 web 確認或刪除。

辨識失敗（沒抓到姓名）的圖會自動移到「我的失敗辨識」（`/contacts/failed-scans`），可在那裡看原圖或刪除；想重試請重新拍照上傳。
