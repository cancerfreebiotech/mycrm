# myCRM ドキュメント

myCRM は CancerFree Biotech 社内で使用する連絡先管理システムです。核心機能は **Telegram Bot で名刺を撮影 → AI 認識 → データベースへ自動保存**であり、Web 管理画面での連絡先メンテナンス・タスク割り当て・レポートエクスポートなどの作業にも対応しています。

---

## システムの特長

| 機能 | 説明 |
|------|------|
| 📷 名刺スキャン | Telegram Bot またはウェブからアップロード。最大 6 枚を合成して AI 認識し、左右対照で確認可能 |
| 👥 連絡先管理 | 完全な連絡先データベース。国フィールド・メールコピー・タグ分類・一括アップロード・Excel エクスポートに対応 |
| ✅ タスク管理 | Bot の自然言語でタスクを作成。Web 画面は 3 タブ構成でアシスタントへの委任も可能 |
| 📝 インタラクションログ | メモ・会議記録・メール履歴を連絡先と紐付け |
| 📧 高度なメール送信 | 宛先編集・テンプレート適用（添付ファイル含む）・AI によるメール本文生成・一時添付ファイルに対応 |
| 📊 レポート | Excel レポートのワンクリックダウンロード、または Gmail への定期自動送信 |
| 🌍 国管理 | 管理者が多言語の国リスト（国旗 emoji 付き）を管理し、連絡先と紐付け |
| 🌐 多言語対応 | 繁体字中国語 / English / 日本語 |
| 🌓 ダークモード | ライト / ダークテーマの切り替えに対応 |
| 📱 モバイル対応 | ハンバーガーサイドメニュー、タブレットではアイコンのみに縮小 |

---

## システムアーキテクチャ

```
Telegram Bot ──→ Webhook (Next.js API Route)
                      │
                      ├──→ Gemini AI (OCR + タスク解析)
                      │
                      └──→ Supabase (PostgreSQL + Storage)
                                │
                          Web Dashboard (Next.js)
                                │
                          Microsoft Teams Bot
```
