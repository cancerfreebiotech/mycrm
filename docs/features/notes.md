# 筆記與互動紀錄

---

## 筆記搜尋

路徑：`/notes`

搜尋所有互動紀錄，支援：
- **關鍵字**：比對紀錄內文
- **日期範圍**：起始日 / 結束日
- **類型篩選**：筆記 / 會議 / Email / 系統
- **分頁**：每頁 20 筆

---

## 未歸類筆記

路徑：`/unassigned-notes`

顯示尚未關聯到任何聯絡人的互動紀錄（`contact_id = null`），例如：
- 在 Bot 輸入 `/note` 但找不到聯絡人時自動建立
- 手動記錄待歸類的備忘

每筆可操作：
- **指定聯絡人**：搜尋並關聯到現有聯絡人
- **刪除**：刪除紀錄

---

## 互動紀錄類型

| 類型 | 說明 | 建立方式 |
|------|------|----------|
| `note` | 文字筆記 | Bot `/note`、Web 手動 |
| `meeting` | 會議記錄 | Bot `/note`（指定 type=meeting） |
| `email` | Email 紀錄 | Bot `/email`、Web 寄信功能 |
| `system` | 系統自動紀錄 | 系統操作（如名片上傳） |

---

## 從 Bot 建立筆記

```
/note 王大華    → 搜尋聯絡人 → 選擇 → 輸入筆記內文
/n @王大華 今天拜訪，對方有意合作   → 快速格式（找到上一個聯絡人時可省略姓名）
```

若找不到對應聯絡人，筆記會存為未歸類，之後可從 Web 歸類。
