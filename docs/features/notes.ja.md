# メモとインタラクションログ

---

## メモ検索

パス：`/notes`

すべてのインタラクション記録を検索できます。対応している検索条件：
- **キーワード**：記録の本文を照合
- **日付範囲**：開始日 / 終了日
- **タイプ絞り込み**：メモ / 会議 / メール / システム
- **ページネーション**：1 ページあたり 20 件

---

## 未割り当てメモ

パス：`/unassigned-notes`

まだ連絡先に紐付けられていないインタラクション記録（`contact_id = null`）を表示します。例：
- Bot で `/note` を使用したが連絡先が見つからなかった場合に自動作成される
- 分類待ちの備忘録として手動記録されたもの

各エントリで操作できること：
- **連絡先を割り当て**：既存の連絡先を検索して紐付ける
- **削除**：記録を削除する

---

## インタラクション記録のタイプ

| タイプ | 説明 | 作成方法 |
|--------|------|---------|
| `note` | テキストメモ | Bot `/note`、Web での手動入力 |
| `meeting` | 会議記録 | Bot `/note`（type=meeting を指定） |
| `email` | メール記録 | Bot `/email`、Web のメール送信機能 |
| `system` | システム自動記録 | システム操作（例：名刺アップロード） |

---

## Bot からのメモ作成

```
/note 山田太郎    → 連絡先を検索 → 選択 → メモ内容を入力
/n @山田太郎 本日訪問、協力に前向き   → クイック形式（前回の連絡先が分かっている場合は名前省略可）
```

対応する連絡先が見つからない場合、メモは未割り当てとして保存され、後で Web から分類できます。
