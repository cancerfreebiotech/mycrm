# 連絡先管理

パス：`/contacts`

---

## 連絡先一覧

### 検索と絞り込み

- **キーワード検索**：名前・メール・会社名を照合（あいまい検索）
- **タグ絞り込み**：タグボタンをクリックすると、そのタグを持つ連絡先のみを表示
- **並び順**：作成日時の降順（最新が先頭）
- **ページネーション**：1 ページあたり 20 件、下部にページナビゲーション

### エクスポート

「Excel エクスポート」または「CSV エクスポート」をクリックすると、現在の絞り込み結果をダウンロードできます（ページネーションの制限なし）。

---

## 連絡先の追加

パス：`/contacts/new`

4 つのセクションに分けて手動入力します：

| セクション | フィールド |
|-----------|-----------|
| **基本情報** | 名前（中国語/英語/現地語）、会社（中国語/英語/現地語）、役職 |
| **連絡先情報** | メール、第 2 メール、電話、第 2 電話、住所、ウェブサイト、**国** |
| **ソーシャルメディア** | LinkedIn、Facebook |
| **備考** | 備考テキスト |

### 名刺 AI 認識（複数枚）

1. 「名刺写真を選択」をクリック — 一度に最大 **6 枚**選択可能（表面・裏面など）
2. 「名刺をスキャン」をクリック — システムが圧縮・合成して AI 認識を実行
3. ページ左側に写真プレビュー、右側に AI 認識結果が並べて表示されます
4. 「フォームに適用」をクリックしてフィールドに入力するか、「無視」をクリックして手動入力

> 保存前にシステムがメールアドレスと名前の類似度を比較して重複連絡先を検出し、確認を促します。

---

## 一括アップロード

パス：`/contacts/batch-upload`

一度に最大 **50 枚**の名刺写真をアップロードできます：

1. クリックまたはドラッグで画像をアップロード（JPG / PNG 対応）
2. システムが最大 5 枚ずつ並行処理します（並行アップロード + AI 認識）
3. プログレスバーで進行状況を確認できます
4. 認識が完了するとプレビューテーブルが表示され、各エントリを個別に編集できます
5. 一括重複検出（既存の連絡先と照合）
6. 「すべて保存」をクリックして一括保存

## 連絡先の詳細

パス：`/contacts/[id]`

### 基本情報セクション

すべての連絡先フィールドを表示します。右上の「編集」ボタンをクリックして編集モーダルを開きます。

- **メール / 電話**フィールドの横にコピーアイコンがあり、クリックでクリップボードにコピーできます
- **国**フィールドには国旗 emoji + 名称が表示されます（例：🇯🇵 日本）

### 名刺管理

1 つの連絡先に複数の名刺を紐付けられます（表面・裏面・カスタムラベルなど）：
- 一度に最大 **6 枚**選択可能。AI が合成して認識し、差分のあるフィールドを確認用に表示します
- カードの右上の 🗑 をクリックして削除します

### インタラクションログのタイムライン

すべての記録を時系列の降順で表示します。タイプ：
- **メモ**（note）
- **会議**（meeting）
- **メール**（email）
- **システム**（system）— システムが自動生成

デフォルトで 20 件を読み込み、下部までスクロールすると自動でさらに読み込みます（無限スクロール）。

### メール送信

「メール送信」ボタンをクリックして高度なメール送信モーダルを開きます：
- **宛先（To）**：編集可能、デフォルトは連絡先のメールアドレス
- **テンプレート**：選択すると件名・本文・添付ファイルが自動入力されます
- **AI 生成**：説明を入力すると、AI が受信者情報と最近のインタラクション記録を基に本文を自動生成します
- **一時添付ファイル**：今回の送信専用の添付ファイルをアップロード（1 ファイル最大 2MB）、テンプレートには保存されません

送信後、メールのインタラクション記録が自動作成されます。

## フィールドの説明

| フィールド | 説明 |
|-----------|------|
| `name` | 中国語名（主表示） |
| `name_en` | 英語名 |
| `name_local` | 現地語名（日本語など） |
| `company` | 会社（中国語） |
| `company_en` | 会社（英語） |
| `company_local` | 会社（現地語） |
| `title` | 役職 |
| `email` | 主メール |
| `second_email` | 第 2 メール |
| `phone` | 主電話 |
| `second_phone` | 第 2 電話 |
| `address` | 住所 |
| `website` | ウェブサイト |
| `linkedin_url` | LinkedIn |
| `facebook_url` | Facebook |
| `notes` | 備考 |
| `country_code` | 国コード（ISO 3166-1 alpha-2、例：`JP`）、countries テーブルに紐付け |
