# 報表

路徑：`/admin/reports`（僅 super_admin 可見）

---

## 立即產生報表

1. 選擇**開始日期**和**結束日期**
2. 點擊「產生報表」
3. 網頁預覽資料表格（新增聯絡人 + 互動紀錄）
4. 點擊「下載 Excel」取得 `.xlsx` 檔案

### Excel 內容

| Sheet | 說明 |
|-------|------|
| **新增名片** | 期間內新增的聯絡人，包含所有欄位 |
| **互動紀錄** | 期間內的所有互動紀錄（筆記/會議/Email） |

---

## Gmail 連結

報表可透過 Gmail 自動寄出，需先完成 OAuth 授權：

1. 點擊「連結 Gmail」按鈕
2. 完成 Google OAuth 授權流程
3. 連結成功後顯示已連結的 Gmail 帳號

> 授權 Token 儲存於 `gmail_oauth` 資料表，逾期自動更新。

---

## 定時排程

設定定期自動寄送報表：

### 新增排程

1. 點擊「+ 新增排程」
2. 填寫：
   - **收件者 Email**
   - **週期**：每週 / 每月 / 自訂 cron 表達式
3. 儲存後排程立即生效

### 排程清單

| 欄位 | 說明 |
|------|------|
| 收件者 | 報表寄送對象 |
| 週期 | 執行頻率 |
| 狀態 | 啟用 / 停用 |
| 操作 | 啟用/停用切換、刪除 |

### cron 表達式範例

| 週期 | cron |
|------|------|
| 每週一早上 9 點 | `0 9 * * 1` |
| 每月 1 日 | `0 9 1 * *` |
| 每天早上 8 點 | `0 8 * * *` |

> 執行時間以 UTC 為基準，台灣時間 = UTC+8，請自行換算。
