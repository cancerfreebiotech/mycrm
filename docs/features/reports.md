# 報表

路徑：`/admin/reports`（僅 super_admin 可見）

---

## 立即產生報表

1. 選擇**開始日期**和**結束日期**
2. 點擊「產生報表」
3. 網頁預覽資料表格（新增聯絡人 + 互動紀錄）
4. 點擊「下載 Excel」取得 `.xlsx` 檔案

### Excel 內容

| Sheet | 說明 |
|-------|------|
| **新增名片** | 期間內新增的聯絡人，包含所有欄位 |
| **互動紀錄** | 期間內的所有互動紀錄（筆記/會議/Email） |

---

## Gmail 授權（自動寄送）

定時排程的報表由後端自動透過 Gmail 寄出，因此需先完成一次 Google 授權。

**此頁面沒有「連結 Gmail」按鈕。** 授權流程需由 super_admin 在登入狀態下，於瀏覽器直接開啟網址 `/api/auth/gmail`：

1. 開啟 `/api/auth/gmail`（會自動導向 Google OAuth 授權畫面）
2. 完成 Google 授權
3. 系統儲存授權憑證（逾期自動更新）

完成後，定時排程即可用該帳號自動寄送報表。此授權通常只需完成一次。

## 定時排程

設定定期自動寄送報表：

### 新增排程

1. 點擊「新增排程」
2. 填寫：
   - **排程名稱**
   - **頻率**：每週 / 每月 / 自訂（Cron）— 選「自訂」時才會出現 Cron 表達式欄位
   - **報表日期範圍（天）**：每次報表涵蓋的天數（1–365）
   - **收件人（逗號分隔）**：可填多位收件人，以逗號分隔
3. 儲存後排程立即生效

> 編輯：點擊排程列的 ✏️ 圖示，會開啟同一表單並帶入現有設定。

### 排程清單

| 欄位 | 說明 |
|------|------|
| 排程名稱 | 排程的名稱 |
| 頻率 | 執行頻率（每週 / 每月 / 自訂） |
| 收件人 | 報表寄送對象 |
| 狀態 | 啟用 / 停用（點擊可切換） |
| 上次執行 | 最近一次自動寄送的時間（台北時間）：成功顯示 ✅、異常顯示 ⚠️；從未執行過顯示「尚未執行」 |
| 操作 | ✏️ 編輯、🗑 刪除 |

### cron 表達式範例

| 週期 | cron |
|------|------|
| 每週一早上 9 點 | `0 9 * * 1` |
| 每月 1 日 | `0 9 1 * *` |
| 每天早上 8 點 | `0 8 * * *` |

> 執行時間以 UTC 為基準，台灣時間 = UTC+8，請自行換算。
