# 儀表板

路徑：`/`（登入後預設頁面）

---

## 統計卡片

儀表板頂部顯示三個即時統計數字：

| 卡片 | 說明 |
|------|------|
| **總聯絡人數** | 系統中所有聯絡人的總筆數 |
| **本月新增** | 本月（1日起）新增的聯絡人數 |
| **未歸類筆記** | 尚未指定聯絡人的互動紀錄數（點擊可進入管理頁） |

---

## Tag 分布圖

以長條圖顯示各 Tag 底下的聯絡人數量，依數量由多到少排列。

> 需要先建立 Tag 才會顯示此區塊。

---

## 國家分布

以長條圖顯示各國家底下的聯絡人數量，依數量由多到少排列，最多顯示前 10 名：

- 每列顯示國旗 emoji、國家名稱與聯絡人數
- 超過前 10 名的國家會合併為「其他國家」
- 沒有填國家的聯絡人歸入「未知」
- 點擊任一列可跳到聯絡人列表並自動套用該國家篩選

> 需要聯絡人已填入國家欄位才會顯示此區塊。

---

## Email 寄送狀態

彙整所有 Email 收件對象目前的可寄送狀態，分為兩部分。

### CRM 聯絡人

以卡片顯示 CRM 聯絡人各種 Email 狀態的數量，點擊任一張卡片可跳到聯絡人列表並套用該狀態篩選：

| 狀態 | 說明 |
|------|------|
| 可寄信 | 目前可正常寄送 |
| 硬退信 | 寄信被永久退回 |
| 無效信箱 | 信箱位址無效 |
| 已退訂 | 對方已取消訂閱 |
| 暫時無法寄送 | 暫時性寄送失敗，稍後可能恢復 |
| 信箱已滿 | 對方信箱空間已滿 |
| 寄件方問題 | 我方（寄件端）造成的寄送問題 |
| 收件方擋信 | 對方伺服器擋信 |

### 外部訂閱者（未連結 CRM）

顯示不在 CRM 聯絡人中的外部電子報訂閱者總數，點擊可進入訂閱名單管理。

---

## 待處理未歸類筆記

列出最新 5 筆未歸類筆記（`contact_id` 為空的互動紀錄），每筆顯示：
- 紀錄類型（筆記 / 會議 / Email / 系統）
- 建立者姓名
- 建立時間
- **指定聯絡人** 按鈕 → 彈出搜尋視窗，可即時歸類

點擊「查看全部」進入完整的 [未歸類筆記頁](notes.md)。
