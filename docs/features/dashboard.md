# 儀表板

路徑：`/`（登入後預設頁面）

---

## 統計卡片

儀表板頂部顯示三個即時統計數字：

| 卡片 | 說明 |
|------|------|
| **總聯絡人數** | 系統中所有聯絡人的總筆數 |
| **本月新增** | 本月（1日起）新增的聯絡人數 |
| **未歸類筆記** | 尚未指定聯絡人的互動紀錄數（點擊可進入管理頁） |

---

## Tag 分布圖

以長條圖顯示各 Tag 底下的聯絡人數量，依數量由多到少排列。

> 需要先建立 Tag 才會顯示此區塊。

---

## 待處理未歸類筆記

列出最新 5 筆未歸類筆記（`contact_id` 為空的互動紀錄），每筆顯示：
- 紀錄類型（筆記 / 會議 / Email / 系統）
- 建立者姓名
- 建立時間
- **指定聯絡人** 按鈕 → 彈出搜尋視窗，可即時歸類

點擊「查看全部」進入完整的 [未歸類筆記頁](notes.md)。
