# 相簿

路徑：`/photos`

相簿以**照片為主體**呈現與聯絡人的合照。一張照片可同時標記多位聯絡人，並會自動出現在每一位被標記聯絡人的詳細頁。

---

## 相簿頁面

以方格網格顯示所有照片，每張照片包含：

- **人數徽章**：右下角以 👥 顯示這張照片已標記的人數（有標記時才出現）
- **聯絡人姓名**：照片下方列出已標記的聯絡人；超過 2 人時顯示「A、B 等 N 人」，尚未標記則顯示「未歸類」
- **拍攝日期**：若照片含 EXIF 拍攝資訊則顯示

### 搜尋與排序

頂部搜尋框可依**附註、地點、聯絡人姓名**關鍵字搜尋。

| 排序 | 說明 |
|------|------|
| **上傳時間** | 預設，依加入相簿的時間 |
| **拍攝時間** | 依照片 EXIF 拍攝日期 |
| **聯絡人名稱** | 依第一位標記聯絡人的姓名 |

---

## 照片如何進入相簿

### Telegram Bot（`/p`）

使用 `/p` 指令，先指定要歸屬的聯絡人：

```
/p                    加到上一位操作過的聯絡人
/p 戴建丞             指定聯絡人（找不到時可直接建立新聯絡人）
/p 戴建丞 | 經濟部    指定聯絡人並帶入公司（也可用逗號分隔）
```

Bot 回覆後，直接把照片傳給 Bot：

1. 可**連續傳送多張**照片，完成後按「完成」
2. 照片會自動壓縮並保留 EXIF（拍攝時間、地點）
3. 傳單張時可再附上一段文字附註

### Web 上傳

在聯絡人詳細頁的「合照」區塊，點「新增合照」即可上傳一或多張照片，系統會自動讀取 EXIF 拍攝時間與地點。

---

## 標記照片中的人

點任一張照片開啟**燈箱（Lightbox）**，右側面板「照片中的人」即為標記管理區。一張照片可標記多位聯絡人。

| 操作 | 步驟 |
|------|------|
| **新增標記** | 點「新增標記」→ 輸入關鍵字搜尋聯絡人 → 點選即完成 |
| **前往聯絡人** | 點已標記的姓名，跳到該聯絡人詳細頁 |
| **移除標記** | 點姓名右側的 ✕ |

> 同一位聯絡人在同一張照片只能標記一次，重複標記會提示「此人已標記在這張照片」。

### AI 建議

若系統辨識出可能的人選，會在面板中以「AI 建議」標示，可點 ✓ **接受**或 ✕ **拒絕**。接受後即成為正式標記。

### 燈箱檢視

- 以滾輪或工具列按鈕縮放，放大後可拖曳平移，雙擊還原
- 側欄顯示拍攝日期、地點、附註與上傳日期
- 按 `ESC` 或點擊背景關閉

---

## 照片如何顯示在聯絡人頁

每一位被**確認**標記的聯絡人，其詳細頁的「合照」區塊都會顯示這張照片。因為一張照片可屬於多人，**同一張合照會同時出現在每位參與者的頁面**。附註可在聯絡人頁點擊照片下方文字編輯。

---

## 刪除與移除

「移除標記」與「刪除照片」是兩件不同的事：

| 操作 | 位置 | 效果 |
|------|------|------|
| **移除標記** | 相簿燈箱「照片中的人」按 ✕ | 只解除該聯絡人與照片的關聯，照片本身與其他人的標記都保留 |
| **刪除照片** | 聯絡人詳細頁「合照」的垃圾桶按鈕 | 永久刪除整張照片（含 Storage 檔案），所有人的標記一併移除 |

> 在相簿中移除某人**不會**刪除這張共用的照片；要真正刪除照片，請到聯絡人詳細頁刪除。
