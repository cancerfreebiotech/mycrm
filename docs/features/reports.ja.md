# レポート

パス：`/admin/reports`（super_admin のみ表示）

---

## レポートをすぐに生成

1. **開始日**と**終了日**を選択します
2. 「レポートを生成」をクリックします
3. ページ上でデータテーブルをプレビューします（新規連絡先 + インタラクション記録）
4. 「Excel をダウンロード」をクリックして `.xlsx` ファイルを取得します

### Excel の内容

| シート | 説明 |
|--------|------|
| **新規名刺** | 期間内に追加された連絡先（すべてのフィールドを含む） |
| **インタラクションログ** | 期間内のすべてのインタラクション記録（メモ/会議/メール） |

---

## Gmail 連携

レポートは Gmail を通じて自動送信できます。事前に OAuth 認可を完了する必要があります：

1. 「Gmail を連携」をクリックします
2. Google OAuth 認可フローを完了します
3. 連携後、接続された Gmail アカウントが表示されます

> 認可トークンは `gmail_oauth` テーブルに保存され、有効期限切れ時に自動更新されます。

---

## 定期スケジュール

定期的なレポートの自動配信を設定します：

### スケジュールの追加

1. 「+ スケジュールを追加」をクリックします
2. 以下を入力します：
   - **受信者メール**
   - **頻度**：毎週 / 毎月 / カスタム cron 式
3. 保存後すぐにスケジュールが有効になります

### スケジュール一覧

| カラム | 説明 |
|--------|------|
| 受信者 | レポートの送信先 |
| 頻度 | 実行頻度 |
| ステータス | 有効 / 無効 |
| 操作 | 有効/無効の切り替え、削除 |

### cron 式の例

| 頻度 | cron |
|------|------|
| 毎週月曜日午前 9 時 | `0 9 * * 1` |
| 毎月 1 日 | `0 9 1 * *` |
| 毎日午前 8 時 | `0 8 * * *` |

> 実行時間は UTC 基準です。日本時間は UTC+9 ですので、ご注意ください。
