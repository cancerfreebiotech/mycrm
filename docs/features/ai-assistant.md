# AI 助理

路徑：`/ai-assistant`

AI 助理是一個對話式介面，用自然語言就能查詢與維護 CRM，不必自己翻頁面。它背後串接聯絡人、電子報名單、標籤與筆記，並可為即將見面的對象排程「會議前情報」。

---

## 在哪裡開啟

| 入口 | 說明 |
|------|------|
| **右下角浮動按鈕** | 每一個頁面右下角都有一顆藍色圓形對話鈕，點擊即從右側滑出對話抽屜，隨時可用 |
| **全頁模式** | 開啟 `/ai-assistant` 進入完整頁面，適合較長的對話 |

兩個入口共用同一個助理，功能完全相同。

---

## 可以做什麼

以自然語言下指令即可，助理會自動挑選需要的動作：

| 類別 | 你可以請它做的事 |
|------|------------------|
| **查詢聯絡人** | 依姓名（任何語言）、email 或公司搜尋；讀出某位聯絡人的完整資料、標籤與最近的互動紀錄 |
| **更新聯絡人** | 修改姓名、公司、職稱、部門、電話、手機、第二 email、地址、LinkedIn／Facebook、認識地點與日期、介紹人、重要程度、語言、醫院等描述性欄位 |
| **加筆記** | 幫聯絡人新增一則互動筆記，可附上會面日期 |
| **標籤** | 列出所有標籤，並幫聯絡人加上或移除標籤 |
| **電子報名單** | 列出所有名單與人數、查看某名單的訂閱者、把某個 email 加入指定名單 |
| **會議前情報** | 為某位聯絡人排程一份「會議前 briefing」，在背景蒐集此人與其公司的最新公開動態；結果稍後在該聯絡人頁面查看 |

助理會用你的語言回覆（預設繁體中文）。

---

## 範例提問

```
搜尋台大的聯絡人
王先生的完整資料和最近互動
把陳小明的職稱改成行銷總監
幫這位聯絡人加一則筆記：今天電話聊到 Q3 合作，下週再約
把這個人標記為重要
列出所有電子報名單
把 alice@example.com 加入「醫師名單」
我下週要跟王先生開會，幫我了解他的近況
```

若一個請求含糊，助理會先幫你搜尋、確認對象，再進行變更。

---

## 寫入操作與安全

助理可以直接改資料，因此設有多層保護：

- **全程留有稽核紀錄**：每一個查詢與變更都會記錄操作者、時間、動作與成功與否，且一律以「你」的身分執行。
- **只能改安全欄位**：更新聯絡人時僅限上表的描述性／關係欄位。**主 email、退訂狀態、名片圖片與系統欄位無法被助理修改。**
- **沒有刪除能力**：助理無法刪除聯絡人、名單或訂閱者。
- **會說明它做了什麼**：進行寫入前，助理會在回覆中說明它變更了哪些內容。由於沒有自動復原，請看過它的說明再繼續。

---

## 限制

- **對話不會保存**：關閉抽屜、重新整理或離開頁面後，這次對話即清空，無法回溯歷史。
- **一次的步驟有上限**：單一請求最多連續進行 6 輪工具操作；太複雜時助理會請你把需求拆小或講得更明確。
- **回覆時間**：每則回覆最長約 60 秒。
- **搜尋筆數**：聯絡人搜尋一次最多 100 筆（預設 20）；名單訂閱者最多 500 筆（預設 50）。
- **更新欄位**：無法變更主 email 與退訂狀態；一次最多異動 50 個欄位。
- **範圍限於既有動作**：助理只能操作聯絡人、名單、標籤、筆記與會議前情報。其他功能（任務、報表、電子報草稿撰寫等）請到各自頁面操作。
