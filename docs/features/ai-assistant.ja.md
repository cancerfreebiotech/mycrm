# AI アシスタント

パス：`/ai-assistant`

AI アシスタントは対話型のインターフェースで、自然言語だけで CRM の照会やメンテナンスができ、自分でページを探し回る必要がありません。裏側で連絡先・ニュースレター名簿・タグ・メモとつながっており、これから会う相手のために「会議前ブリーフィング」を予約することもできます。

---

## 開く場所

| 入口 | 説明 |
|------|------|
| **右下のフローティングボタン** | すべてのページの右下に青い丸型のチャットボタンがあります。タップすると右側からチャットドロワーがスライド表示され、いつでも使えます |
| **フルページモード** | `/ai-assistant` を開くと完全なページになり、長めの会話に向いています |

2 つの入口は同じアシスタントを共有し、機能はまったく同じです。

---

## できること

自然言語で指示するだけで、アシスタントが必要なアクションを自動で選びます：

| カテゴリ | 依頼できること |
|----------|----------------|
| **連絡先の照会** | 名前（どの言語でも）・メール・会社で検索。ある連絡先の詳細情報・タグ・直近のやり取り履歴を読み上げます |
| **連絡先の更新** | 名前、会社、役職、部署、電話、携帯、第 2 メール、住所、LinkedIn／Facebook、出会った場所と日付、紹介者、重要度、言語、病院などの説明的なフィールドを変更します |
| **メモの追加** | 連絡先にやり取りのメモを 1 件追加します。面談日を付けることもできます |
| **タグ** | すべてのタグを一覧表示し、連絡先へのタグの追加・削除を行います |
| **ニュースレター名簿** | すべての名簿と人数を一覧表示し、ある名簿の購読者を確認し、指定した名簿にメールを追加します |
| **会議前ブリーフィング** | ある連絡先の「会議前ブリーフィング」を予約し、その人と会社の最新の公開情報をバックグラウンドで収集します。結果はあとでその連絡先のページで確認します |

アシスタントはあなたの言語（デフォルトは繁体字中国語）で返信します。

---

## 質問例

```
台大の連絡先を検索
王さんの詳細情報と直近のやり取り
陳小明さんの役職をマーケティング部長に変更
この連絡先にメモを追加：本日電話で Q3 の協業について話した、来週再度アポ
この人を重要としてマーク
すべてのニュースレター名簿を一覧表示
alice@example.com を「医師名簿」に追加
来週王さんと会うので、近況を把握させて
```

依頼があいまいな場合、アシスタントはまず検索して対象を確認してから変更を行います。

## 書き込み操作と安全性

アシスタントはデータを直接変更できるため、複数の保護策が設けられています：

- **すべて監査記録が残ります**：すべての照会と変更について、操作者・時刻・アクション・成否が記録され、常に「あなた」の名義で実行されます。
- **安全なフィールドのみ変更可能**：連絡先の更新時は、上表の説明的／関係的なフィールドに限られます。**主メール、配信停止ステータス、名刺画像、システムフィールドはアシスタントでは変更できません。**
- **削除はできません**：アシスタントは連絡先・名簿・購読者を削除できません。
- **何をしたか説明します**：書き込みの前に、アシスタントは返信の中で何を変更したかを説明します。自動的な取り消しはないため、その説明を確認してから続けてください。

---

## 制限

- **会話は保存されません**：ドロワーを閉じる、再読み込みする、ページを離れると、その会話はクリアされ、履歴をたどることはできません。
- **1 回あたりのステップに上限があります**：1 つの依頼で連続して実行されるツール操作は最大 6 ラウンドです。複雑すぎる場合、アシスタントは依頼を小さく分けるか、より明確にするよう求めます。
- **応答時間**：1 回の返信は最長で約 60 秒かかります。
- **検索件数**：連絡先検索は最大 100 件（デフォルト 20 件）、名簿の購読者は最大 500 件（デフォルト 50 件）です。
- **更新フィールド**：主メールと配信停止ステータスは変更できません。1 回の更新で変更できるフィールドは最大 50 件です。
- **既存のアクションに限定**：アシスタントが操作できるのは連絡先・名簿・タグ・メモ・会議前ブリーフィングのみです。その他の機能（タスク、レポート、ニュースレター下書きの作成など）は各ページで操作してください。
