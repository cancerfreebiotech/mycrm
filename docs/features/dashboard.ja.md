# ダッシュボード

パス：`/`（ログイン後のデフォルトページ）

---

## 統計カード

ダッシュボード上部にリアルタイムの統計数字が 3 つ表示されます：

| カード | 説明 |
|--------|------|
| **総連絡先数** | システム内の連絡先の合計件数 |
| **今月の新規追加** | 今月（1 日から）追加された連絡先数 |
| **未割り当てメモ** | 連絡先に紐付けられていないインタラクション記録の件数（クリックで管理ページへ移動） |

---

## タグ分布グラフ

各タグの連絡先数を棒グラフで表示し、件数の多い順に並べます。

> タグを作成してからこのセクションが表示されます。

---

## 国別分布

各国の連絡先数を棒グラフで表示し、件数の多い順に並べます（最大で上位 10 か国まで表示）：

- 各行に国旗 emoji・国名・連絡先数を表示
- 上位 10 か国を超える国は「その他の国」にまとめられます
- 国が未入力の連絡先は「不明」に分類されます
- 各行をクリックすると、その国で絞り込んだ連絡先一覧へ移動します

> 連絡先に国が入力されている場合にこのセクションが表示されます。

---

## メール送信状況

すべてのメール送信先の現在の送信可否状況を、2 つに分けて集計します。

### CRM 連絡先

CRM 連絡先の各メール状態の件数をカードで表示します。カードをクリックすると、その状態で絞り込んだ連絡先一覧へ移動します：

| 状態 | 説明 |
|------|------|
| メール送信可 | 現在、正常に送信可能 |
| バウンス | 送信が恒久的に返送された |
| 無効なメール | アドレスが無効 |
| 配信停止 | 相手が配信を停止した |
| 一時的な送信失敗 | 一時的な送信失敗（後で回復する可能性あり） |
| メールボックス満杯 | 相手のメールボックスが満杯 |
| 送信側の問題 | こちら（送信側）に起因する送信問題 |
| 受信側ブロック | 相手のサーバーにブロックされた |

### 外部購読者（CRM 連携なし）

CRM 連絡先に含まれない外部ニュースレター購読者の総数を表示します。クリックすると購読リスト管理へ移動します。

---

## 未処理の未割り当てメモ

未割り当てメモ（`contact_id` が null のインタラクション記録）の最新 5 件を一覧表示します。各エントリには以下が表示されます：
- 記録タイプ（メモ / 会議 / メール / システム）
- 作成者名
- 作成日時
- **連絡先を割り当て**ボタン → 検索ポップアップを開き、すぐに割り当て可能

「すべて表示」をクリックすると [未割り当てメモページ](notes.md) へ移動します。
