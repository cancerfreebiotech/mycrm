# 個人設定

パス：`/settings`

---

## プロフィール

| フィールド | 説明 |
|-----------|------|
| 表示名 | システムで表示される名前 |
| メール | ログインに使用する Microsoft メール（読み取り専用） |
| ロール | member / super_admin（読み取り専用、管理者が設定） |
| Telegram ID | Telegram Bot との連携に使用 |

---

## 外観設定

### テーマ
- **ライトモード** ☀️
- **ダークモード** 🌙

ヘッダー右上の ☀️/🌙 ボタンでも素早く切り替えられます。

### 言語
- 繁体字中国語
- English
- 日本語

## AI モデルの選択

個人で使用する AI モデルを選択します（名刺認識・メール生成などに使用）：

1. まず **AI プロバイダー**（Endpoint）を選択します
2. 次にそのプロバイダー配下の**モデル**を選択します

利用可能なプロバイダーとモデルは、super_admin が [AI モデル管理](../admin/models.md) ページで管理します。

---

## 私のアシスタント

マネージャーは 1 名以上の**アシスタント**を設定できます。アシスタントはマネージャーの代わりに以下を行えます：
- タスクを完了としてマーク
- マネージャーが作成したタスクを操作

### アシスタントの追加

1. 「アシスタントを追加」フィールドにアシスタントのメールアドレスを入力します
2. 「追加」をクリックします

### アシスタントの削除

アシスタント一覧の 🗑 アイコンをクリックして削除します。

> 注意：アシスタントはシステムの既存メンバー（`users` テーブルに登録済み）である必要があります。
