# myCRM 説明書

myCRM は cancerfree.io の全社で使用する顧客関係管理システムです。基本の流れは、Telegram Bot で名刺を撮影し、AI が自動で認識し、ワンタップで CRM に保存するというもので、その後は Web インターフェースで連絡先・メモ・タスク・レポート・ニュースレターをまとめて管理できます。

> 現在のバージョン：**v7.7.0**

## セクション案内

- **[はじめに](getting-started/first-login.md)** — 初回ログイン、MFA（二段階認証）の設定、システムの導入。
- **[機能説明](features/dashboard.md)** — ダッシュボード、連絡先、タスク、メモ、レポート、ニュースレター、Social Briefing、AI アシスタント、アルバムなどの操作説明。
- **[Telegram Bot](bot/overview.md)** — Bot 概要、コマンド一覧、設定ガイド。
- **[Microsoft Teams Bot](bot/teams-overview.md)** — Teams のタスク通知と会議スケジュール設定。
- **[管理者](admin/users.md)** — ユーザー、タグ、AI モデル、レポートの管理。

各バージョンの詳細な変更点は[変更履歴](CHANGELOG.md)を参照してください。
