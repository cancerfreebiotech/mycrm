# myCRM 說明書

myCRM 是 cancerfree.io 全公司使用的客戶關係管理系統。核心流程是用 Telegram Bot 拍名片、由 AI 自動辨識，再一鍵存入 CRM，之後即可在 Web 介面統一管理聯絡人、筆記、任務、報表與電子報。

> 目前版本：**v7.2.8**
