## myCRM とは

myCRM は cancerfree.io 全社で共用する名刺・顧客管理システムです。基本の流れはシンプルで、Telegram で名刺を撮影 → AI が氏名・会社・役職・メールなどの項目を自動抽出 → CRM に保存されます。その後は Web 画面で連絡先の検索・管理、訪問やメモの記録、タスクの割り当て、Email マーケティングの配信ができます。このページでは、初回ログインから使い始めるまでの手順を順に説明します。

## Microsoft アカウントでサインイン

myCRM は **Microsoft アカウント（Azure AD）シングルサインオン（SSO）**を使用しており、`@cancerfree.io` ドメインの会社アカウントのみ利用できます。別途パスワードを設定する必要はありません。

1. ブラウザで `crm.cancerfree.io` にアクセスします
2. **「Microsoft アカウントでサインイン」**をクリックします
3. 会社のメールアドレスとパスワードを入力し、アプリケーションの認可に同意します
4. 初回ログインで自動的にアカウントが作成され（ロールのデフォルトは `member`）、続けて MFA の設定を求められます

> 「@cancerfree.io アカウントのみサインイン可能」と表示された場合は、会社以外のアカウントを使用しています。会社のメールアドレスで再度サインインしてください。

## MFA（二段階認証）の設定と利用

アカウントのセキュリティのため、**すべてのユーザーは MFA（二段階認証）の設定が必須**です。設定を完了するまでどの機能も利用できません。

### 初回の MFA 設定

初回ログイン後、システムは自動的に MFA 設定ページに遷移します：

1. スマートフォンに認証アプリ（Google Authenticator、Microsoft Authenticator、1Password など、どれでも可）をインストールします
2. 画面の **QR コードをアプリでスキャン**します。スキャンできない場合は、コードの下に表示される**手動キー**を入力してください
3. アプリが生成する **6 桁の認証コード**を入力して登録を完了します
4. 登録が完了すると、システムを通常どおり利用できます

### 以降のログインごと

登録後は、ログインのたびに認証アプリに表示される現在の **6 桁コード**の入力が求められます。コードは 30 秒ごとに更新されるため、最新のものを入力してください。

### スマートフォンの変更・紛失時

- 端末を変更・再登録する場合：画面右上 → **個人設定** → MFA セクションで管理します。
- スマートフォンの紛失・機種変更などでコードを取得できない場合：Super Admin に連絡し、**ユーザー管理**ページで MFA をリセットしてもらってください。リセット後、再度ログインして新しい端末を登録します。

## 画面の説明

ログイン後、左側にサイドバーが表示されます。上部はすべてのメンバーが使用できるコア機能、区切り線より下は権限付与が必要な高度な機能と Super Admin 専用項目です。各ページの右下には **AI アシスタント**のフローティングボタンがあり、どのページからでも開けます。

### コア機能（すべてのメンバーが利用可能）

| 項目 | 説明 |
|---|---|
| ダッシュボード | ダッシュボードと統計 |
| 連絡先 | 名刺連絡先一覧 |
| 確認待ち名刺 | AI 認識後、確認待ちの名刺 |
| 認識失敗 | 自分がスキャンして認識に失敗した名刺 |
| アルバム | 名刺・イベント写真 |
| メモ検索 | 連絡先メモの検索 |
| タスク | タスク一覧 |
| レポート | 統計レポート |
| メールキャンペーン | Email マーケティング |
| 個人設定 | プロフィール・言語・テーマ・MFA・Telegram 連携 |
| 使用説明書 | 本ドキュメント |
| 💬 フィードバック | フィードバック送信 |

### 高度な機能（Super Admin による機能ごとの権限付与が必要）

これらの項目はサイドバーに**常に表示**されますが、利用するには Super Admin が**ユーザー管理**ページで機能ごとに権限を付与する必要があります。権限がない状態でページを開くと、権限不足の案内が表示されます：タグ管理、未分類メモ、メールテンプレート、プロンプト管理、国管理、ニュースレター、スキャン失敗、重複連絡先、共用 Email、Email リカバリー、CamCard 取込、ゴミ箱、ユーザー管理。

### Super Admin 専用

Super Admin のみに表示：組織設定、AI モデル管理、システム健全性、MCP Tokens、MCP 活動、監査ログ、配信停止管理、DSAR 照会、ニュースレター概要、フィードバック管理。

### サイドバーの操作

- **モバイル**：左上のハンバーガーメニューでサイドバーを開閉します。
- **デスクトップ**：マウスオーバーで展開し、左下の矢印で折りたたみを固定できます。
- タッチ操作の対象はすべて 44px 以上で、スマートフォンでの片手操作に配慮しています。

## 言語とテーマ

### 言語の切り替え（繁体字中国語 / English / 日本語）

どちらの方法でも切り替えられます。設定はアカウントに保存され、次回ログイン時に自動的に適用されます：

- ヘッダー右上の 🌐 アイコンをクリック → 言語を選択
- **個人設定** → 言語選択

### ダーク / ライトテーマ

- ヘッダー右上の ☀️ / 🌙 アイコンでテーマを即時に切り替えられます
- **個人設定** → テーマ選択でも設定できます
- デフォルトはライト（Light）で、ダークに切り替えられます。設定した内容は記憶されます

## Telegram Bot の連携

Telegram で名刺撮影・メモ・タスク管理を行うには、先に Telegram アカウントを連携します。仕組みは、**Telegram の数字 ID** を個人設定に入力しておくと、Bot がメッセージを受け取った際にその ID であなたを識別する、というものです。

### ステップ 1：Telegram ID を取得する

Telegram で **`@userinfobot`** を検索し、任意のメッセージを送信すると、数字の ID（例：`123456789`）が返されます。

### ステップ 2：個人設定に入力する

myCRM にログイン → 画面右上 → **個人設定** → Telegram ID を貼り付けて保存します。ID は数値である必要があります。「他のメンバーに既に登録されています」と表示された場合、その ID はすでに使用されています。入力ミスがないか確認してください。

### ステップ 3：Bot を使い始める

Telegram で **`@CF_CRMBot`**（CancerFreeCRM）を検索して会話を開始します。**`/help`** を送信すると利用可能なコマンド一覧が表示されます。名刺の写真をそのまま送れば認識をテストできます。Bot が「まだ認可されていません」と返す場合、ID が正しく連携されていません。個人設定に戻って確認してください。

> タスク通知を受け取りたい場合は、Microsoft Teams Bot も連携できます：Teams で Bot との 1 対 1 チャットを開き、任意のメッセージ（例：`help`）を送信すると、Bot が会社アカウントで自動的に連携します。以降、タスクが割り当てられると通知カードが自動的に届きます。

## 困ったときは？フィードバックの送信

サイドバーの **💬 フィードバック**（`/feedback`）から、問題や要望をいつでもチームに報告できます：

1. 種類を選択：**機能リクエスト** または **バグ報告**
2. タイトルと詳細説明を入力（いずれも必須）
3. 必要に応じてスクリーンショットを 1 枚添付（バグ報告では推奨）
4. 送信後、管理者が管理画面の**フィードバック管理**で追跡・対応します

## ロールと権限

myCRM のロールは 2 種類のみです：

| ロール | 説明 |
|---|---|
| `member` | 一般ユーザー。すべてのコア機能を使用できます。高度な機能には別途権限付与が必要です。 |
| `super_admin` | システム管理者。すべての機能を備え、ユーザー管理、権限の付与、他ユーザーの MFA リセット、AI モデルやシステム設定の管理ができます。 |

高度な機能は**機能ごとの権限付与**で制御されます：Super Admin が**ユーザー管理**ページで個々のメンバーに必要な機能（例：ニュースレター、タグ管理など）を有効化すると、そのメンバーはロールを変更せずにその機能を使用できます。
