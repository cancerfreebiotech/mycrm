## 什麼是 myCRM

myCRM 是 cancerfree.io 全公司共用的名片與客戶關係管理系統。核心流程是：用 Telegram 拍下名片 → AI 自動辨識姓名、公司、職稱、Email 等欄位 → 存進 CRM。之後可在網頁介面搜尋與管理聯絡人、記錄拜訪與筆記、指派任務、寄送 Email 行銷活動。本頁帶你完成從第一次登入到開始使用的所有步驟。

## 使用 Microsoft 帳號登入

myCRM 採 **Microsoft 帳號（Azure AD）單一登入（SSO）**，僅限 `@cancerfree.io` 網域的公司帳號，不需另設密碼。

1. 用瀏覽器前往 `crm.cancerfree.io`
2. 點擊 **「使用 Microsoft 帳號登入」**
3. 輸入公司 Email 與密碼，並同意應用程式授權
4. 首次登入會自動建立你的帳號（角色預設為 `member`），接著要求完成 MFA 設定

> 若看到「僅限 @cancerfree.io 帳號登入」，代表你用了非公司帳號，請改用公司 Email 重新登入。

## 設定與使用 MFA（雙重驗證）

為保護帳號安全，**所有使用者都必須完成 MFA（雙重驗證）**；未完成前無法使用任何功能。

### 首次設定 MFA

首次登入後系統會自動導向 MFA 設定頁：

1. 在手機安裝任一驗證器 App（Google Authenticator、Microsoft Authenticator、1Password 等皆可）
2. 用 App **掃描畫面上的 QR Code**；若無法掃描，改用畫面上顯示的**手動金鑰**輸入
3. 輸入 App 產生的 **6 位數驗證碼**完成綁定
4. 綁定成功後即可正常使用系統

### 之後每次登入

綁定完成後，每次登入都會要求輸入驗證器當下顯示的 **6 位數驗證碼**才能進入。驗證碼每 30 秒更新一次，請輸入最新的一組。

### 更換手機或遺失驗證器

- 想更換裝置或重新綁定：右上角 → **個人設定** → MFA 區塊管理。
- 手機遺失、換機或無法取得驗證碼：請聯絡 Super Admin，於 **使用者管理** 頁面為你重設 MFA；重設後重新登入即可綁定新裝置。

## 介面導覽

登入後左側是側邊欄。上半部為所有成員都能使用的核心功能，分隔線以下為需授權的進階功能與 Super Admin 專屬項目。頁面右下角有 **AI 助理**浮動按鈕，任何頁面都可點開詢問。

### 核心功能（所有成員可用）

| 項目 | 說明 |
|---|---|
| 總覽 | 儀表板與統計 |
| 聯絡人 | 名片聯絡人清單 |
| 待審核名片 | AI 辨識後待你確認的名片 |
| 我的失敗辨識 | 你自己掃描但辨識失敗的名片 |
| 相簿 | 名片與活動照片 |
| 筆記搜尋 | 搜尋聯絡人筆記 |
| 任務 | 任務清單 |
| 報表 | 統計報表 |
| 郵件活動 | Email 行銷活動 |
| 個人設定 | 個人資料、語言、主題、MFA、Telegram 綁定 |
| 使用說明 | 本說明文件 |
| 💬 回饋 | 意見回饋 |

### 進階功能（需 Super Admin 逐項授權）

這些項目在側邊欄**一律可見**，但需 Super Admin 在**使用者管理**中逐項授權後才能使用——未授權者進入頁面會看到權限不足提示：Tag 管理、未歸類筆記、郵件範本、Prompt 管理、國家管理、電子報、失敗掃描、重複聯絡人、共用 Email、Email 復活、CamCard 匯入、回收區、使用者管理。

### Super Admin 專屬

僅 Super Admin 可見：組織設定、AI 模型管理、系統健康、MCP Tokens、MCP 活動、稽核日誌、退訂管理、資料查詢（DSAR）、電子報總覽、回饋管理。

### 側邊欄操作

- **手機**：以左上角漢堡選單開關側邊欄。
- **桌機**：滑鼠移入自動展開，左下角箭頭可固定收合。
- 觸控目標皆 ≥ 44px，適合手機單手操作。

## 語言與主題

### 切換語言（繁中 / English / 日本語）

兩種方式皆可，設定會存到你的帳號、下次登入自動套用：

- Header 右上角點 🌐 圖示 → 選擇語言
- **個人設定** → 語言選擇

### 深色 / 淺色主題

- Header 右上角的 ☀️ / 🌙 圖示可即時切換深淺色
- 也可在 **個人設定** → 主題選擇設定
- 預設為淺色（Light），可切換深色；你的偏好會被記住

## 綁定 Telegram Bot

要用 Telegram 拍名片、記筆記、管理任務，需先綁定 Telegram 帳號。原理是：把你的 **Telegram 數字 ID** 填進個人設定，Bot 收到訊息時會以這個 ID 認出你。

### 步驟 1：取得你的 Telegram ID

在 Telegram 搜尋 **`@userinfobot`**，發送任意訊息，它會回傳你的數字 ID（例如 `123456789`）。

### 步驟 2：填入個人設定

登入 myCRM → 右上角 → **個人設定** → 將 Telegram ID 貼上並儲存。ID 必須為數字；若顯示「已被其他成員綁定」，代表該 ID 已被使用，請確認是否填錯。

### 步驟 3：開始使用 Bot

在 Telegram 搜尋 **`@CF_CRMBot`**（CancerFreeCRM）並開始對話。傳送 **`/help`** 可看到所有可用指令；直接傳一張名片照片即可測試辨識。若 Bot 回覆「尚未獲得授權」，代表 ID 尚未正確綁定，請回個人設定確認。

> 想接收任務通知，也可另外綁定 Microsoft Teams Bot：在 Teams 開啟與 Bot 的 1 對 1 聊天並傳送任意訊息（如 `help`），Bot 會自動以你的公司帳號完成綁定，日後任務指派會自動送出通知卡片。

## 遇到問題？提交回饋

側邊欄的 **💬 回饋**（`/feedback`）可隨時把問題或需求回報給團隊：

1. 選擇類型：**新功能需求** 或 **Bug 回報**
2. 填寫標題與詳細說明（皆必填）
3. 需要時附上一張截圖（Bug 回報建議附上）
4. 送出後由管理員在後台**回饋管理**追蹤與處理

## 角色與權限

myCRM 只有兩種角色：

| 角色 | 說明 |
|---|---|
| `member` | 一般使用者，可用所有核心功能；進階功能需另外授權。 |
| `super_admin` | 系統管理員，具備全部功能，並可管理使用者、指派授權、重設他人 MFA、管理 AI 模型與系統設定。 |

進階功能採**逐項授權**：Super Admin 在**使用者管理**為個別成員開啟所需功能（如電子報、Tag 管理等），成員即可使用，無需變更角色。
